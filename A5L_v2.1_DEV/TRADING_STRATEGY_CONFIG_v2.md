## 🇨🇳 A股策略详解 - 超短轮动

### 核心思想
**避免长时间持股暴露，抓短期热点，快速轮动**

### 入场条件
1. ✅ 成交量突增（3日均量2倍以上）
2. ✅ 价格突破5日新高
3. ✅ 动量排名进入前20%
4. ✅ 近期无涨停（避免追高）

### 出场条件
1. 🛑 止损：-3%（严格止损）
2. 🎯 止盈：+8%（见好就收）
3. ⏰ 时间：最多持仓3天
4. 📉 动量衰减：跌破-0.5%
5. 📊 缩量：低于2日均量70%
6. 🚨 黑天鹅：市场跌5%立即清仓

### 仓位管理
- 单股最大仓位：8%
- 最多持股：8只
- 现金比例最低：30%（随时可切换）
- 日最大交易：8次
- 板块轮动频率：每日

### 轮动逻辑
```
热点板块A ──► 发现新热点B ──► 卖出A买入B
   │                                    │
   └──────────── 循环轮动 ──────────────┘
```

## 🇺🇸 美股策略详解 - 大波段

### 核心思想
**抓大趋势，长期持有，让利润奔跑**

### 入场条件
1. ✅ 20日均线上穿60日均线（金叉）
2. ✅ 价格突破60日新高
3. ✅ 成交量高于20日均量30%
4. ✅ 趋势强度>0.6

### 出场条件
1. 🛑 止损：-8%（宽松止损）
2. 🎯 止盈：+25%（让利润奔跑）
3. 📈 追踪止损：回撤12%
4. 📉 死叉：20日均线下穿60日
5. 🚨 黑天鹅：VIX>40或标普跌5%清仓

### 仓位管理
- 单股最大仓位：20%
- 最大总仓位：70%
- 日最大交易：2次
- 板块集中度限制：35%

### 持有风格
```
买入 ──► 持有30天 ──► 趋势延续则继续持有
            │
            ▼
    趋势反转或达到目标 ──► 卖出
```

## 🇭🇰 港股策略详解 - 波段价值

### 核心思想
**低估值买入，趋势确认后持有，价值+波段结合**

### 入场条件
1. ✅ PE < 12（低估值）
2. ✅ PB < 1.2（破净或接近破净）
3. ✅ 股息率 > 4%
4. ✅ 价格站上60日均线
5. ✅ 趋势开始向上

### 出场条件
1. 🛑 止损：-6%
2. 🎯 止盈：+20%
3. 📈 追踪止损：回撤10%
4. 📊 估值修复：PE>18或PB>2
5. ⏰ 时间：最多60天
6. 🚨 黑天鹅：恒指跌8%清仓

### 仓位管理
- 单股最大仓位：18%
- 最大总仓位：65%
- 日最大交易：3次

### 持有风格
```
低估值买入 ──► 等待估值修复 ──► 趋势确认持有
                    │
                    ▼
    估值合理或趋势反转 ──► 卖出
```

## 🛡️ 黑天鹅风控机制

### 监控指标

| 触发器 | 条件 | 动作 | 优先级 |
|--------|------|------|--------|
| 美股崩盘 | VIX>40 且 标普<-5% | 🚨 全仓清仓 | P0 |
| A股千股跌停 | 沪深300<-5% 且 跌停>500 | ⚠️ 减仓至30% | P1 |
| 港股暴跌 | 恒指<-8% 且 放量 | 🚨 全仓清仓 | P0 |
| 跨市场联动 | 相关性>0.9 且 恐慌>80 | ⚠️ 全市场减仓50% | P1 |
| 个股黑天鹅 | 单日跌>20% | 🛑 立即止损 | P0 |

### 执行原则
1. **快速反应**: 检测到极端情况后5分钟内执行
2. **果断斩仓**: 不犹豫，不幻想，机械执行
3. **保护本金**: 生存第一，盈利第二
4. **事后复盘**: 每次触发后分析原因，优化模型
