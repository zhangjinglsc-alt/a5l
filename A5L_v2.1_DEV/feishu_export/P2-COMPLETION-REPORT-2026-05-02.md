## 🎯 任务4: 多因子模型

### Barra因子体系 (20+因子)

| 类别 | 因子 | 说明 |
|------|------|------|
| 规模 | size, size_nonlinear | 市值对数、非线性 |
| 价值 | BP, EP, SP | 账面市值比、盈利市值比 |
| 成长 | earnings_growth, sales_growth | 盈利/营收增长 |
| 质量 | ROE, ROA | 净资产/总资产收益率 |
| 动量 | momentum_1m/3m/6m/12m | 多期动量 |
| 波动 | volatility, beta | 波动率、贝塔 |
| 流动性 | turnover, volume_ratio | 换手率、量比 |
| 技术 | RSI, MACD, Bollinger | 技术指标 |

### 因子检验
- IC计算 (Spearman秩相关)
- IR比率 (信息比率)
- T检验 + P值
- 正IC比例

### 因子组合
- 有效因子筛选 (|IC|>0.02, IR>0.3, p<0.05)
- 3种加权方法:
  - 等权
  - IC加权
  - 风险平价
- 综合因子得分

### 演示结果
```

Top 3: max_drawdown_6m (IC=-0.53), size (IC=-0.48), BP (IC=+0.48)
综合得分: 均值-0.0000, 标准差0.5070
```
