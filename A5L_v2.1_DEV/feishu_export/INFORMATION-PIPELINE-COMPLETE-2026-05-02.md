## 🎯 核心功能

### 1. 阅读信息 (Step 1)
```python
- 自动分类: market_data / financial_report / trading_strategy / news
- 关键词提取
- 内容长度统计
```

### 2. 复查确认 (Step 2)
```python
- 来源可信度评估:
  * 官方公告: 5分
  * 交易所: 5分
  * 财经媒体: 3分
  * 社交媒体: 2分
  
- 质量检查:
  * 是否包含具体数据
  * 是否有来源引用
  * 时效性评估
  
- 可靠性等级:
  * VERIFIED (已验证)
  * HIGH (高可靠)
  * MEDIUM (中等)
  * LOW (低可靠)
  * UNVERIFIED (未验证)
```

### 3. 分析+KIWI (Step 3)
```python
- SKILL分析:
  * 实体识别 (股票代码、公司名称)
  * 情感分析 (偏多/偏空/中性)
  * 潜在影响评估
  * 相关策略匹配
  
- KIWI调阅:
  * 自动搜索相关知识
  * 关联历史文档
  * 交叉验证信息
```

### 4. 输出理解 (Step 4)
```python
输出结构:
- 【信息来源】
- 【可靠性】等级 + 评分
- 【内容摘要】
- 【情感倾向】
- 【潜在影响】
- 【关键洞察】
- 【建议行动】
```

### 5. 归档总结 (Step 5)
```python
- 可靠性≥60%: 归档到KIWI
- 策略相关: 更新策略参数
- 形成处理闭环记录
```
