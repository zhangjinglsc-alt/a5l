## 💡 核心能力

### 1. 默契度评分系统 (0-100分)

| 分数 | 等级 | 表情 | 说明 |
|------|------|------|------|
| 80-100 | 灵魂伴侣 | 💕 | 高度默契，能预测用户需求 |
| 60-79 | 亲密战友 | 🤝 | 很好的配合，了解主要习惯 |
| 40-59 | 默契搭档 | 👥 | 基本了解，配合顺畅 |
| 20-39 | 熟悉朋友 | 👋 | 初步了解，正在磨合 |
| 0-19 | 初识之交 | 🌱 | 刚开始建立关系 |

**评分维度**:
- 交互次数 (30分)
- 偏好设置完整度 (20分)
- 行为模式识别 (30分)
- 使用时长 (20分)

### 2. 学习维度

**沟通偏好**:
- 回复风格 (简洁/详细/幽默/专业)
- 回复长度 (短/中/长)
- 表情符号使用
- Markdown表格使用

**时间偏好**:
- 每日分析时间 (默认09:00)
- 每日复盘时间 (默认21:00)
- 免打扰时段

**投资偏好**:
- 关注市场 (A股/美股/港股)
- 偏好策略 (趋势/价值/超短线)
- 风险承受能力
- 持仓周期

**报告偏好**:
- 报告格式 (Markdown/JSON/CSV)
- 报告内容章节
- 是否包含图表

### 3. 核心功能

| 功能 | 方法 | 说明 |
|------|------|------|
| 行为记录 | `record_action()` | 自动记录用户每次操作 |
| 偏好学习 | `update_preference()` | 学习用户偏好设置 |
| 模式识别 | `_update_patterns()` | 识别行为模式 |
| 默契评分 | `_update_rapport_score()` | 计算默契度 |
| 个性问候 | `get_personalized_greeting()` | 定制问候语 |
| 智能建议 | `suggest_next_actions()` | 预测下一步需求 |
| 关注追踪 | `get_stock_interests()` | 追踪关注股票 |

## 🎮 使用示例

### 示例1: 记录用户行为
```python
from skills.ARCHITECT-5L-SUPER.SKILL import Architect5LSuperSkill

skill = Architect5LSuperSkill()

# 记录用户分析股票的行为
skill.record_user_action(
    action_type="analyze_stock",
    details={"symbol": "300308.SZ", "market": "A股"},
    context="分析CPO龙头"
)
```

### 示例2: 获取默契度
```python
# 查看当前默契度
rapport = skill.get_user_rapport_score()
print(f"默契度: {rapport['score']:.1f}/100")
print(f"关系等级: {rapport['level']}")
```

**输出**:
```
默契度: 30.8/100
关系等级: 熟悉朋友
```

### 示例3: 个性化问候
```python
# 获取根据默契度定制的问候
greeting = skill.get_personalized_greeting()
print(greeting)
```

**输出示例**:
- 低默契度: "下午好！我是A5L，让我们一起建立默契吧！🚀"
- 中默契度: "下午好！我们的默契度正在提升中(31分)，越来越了解你了！💪"
- 高默契度: "下午好！我们的默契度已经达到亲密战友级别(68分)了！🎉"

### 示例4: 智能建议
```python
# 获取下一步操作建议
suggestions = skill.suggest_next_actions()

for suggestion in suggestions:
    print(f"💡 {suggestion['description']}")
    print(f"   原因: {suggestion['reason']}")
```

### 示例5: 查看关注股票
```python
# 获取用户关注的股票 (基于历史行为)
interests = skill.get_user_stock_interests()

for stock in interests[:5]:
    print(f"📈 {stock['symbol']} - 关注{stock['interest_score']}次")
```

**输出**:
```
📈 300308.SZ - 关注2次
📈 000977.SZ - 关注1次
📈 688041.SH - 关注1次
```
