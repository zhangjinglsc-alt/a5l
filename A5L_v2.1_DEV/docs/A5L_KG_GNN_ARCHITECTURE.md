## 🎯 三大核心技术详解

### 1. 时序图卷积 (Temporal GCN)

**核心问题**: 传统GNN处理静态图，金融市场是流动的

**技术方案**:
```python
# 1. 时间切片编码
for t in [T-2, T-1, T]:
    snapshot = encode_temporal_snapshot(kg_at_time_t)
    # 使用R-GCN编码结构特征

# 2. 演化单元更新
evolutionary_embedding[t] = GRU(
    current_structural_features,
    evolutionary_embedding[t-1]
)

# 3. 时序预测
prediction = predict_relation_transfer(
    source, relation, target, 
    future_days=3
)
```

**实战效果**:
```
场景: 源杰科技(光芯片)股价异动 +15%
传统KG: 找到静态关联"中际旭创"
Temporal KG-NN: 
  - 历史模式: "上游异动3天后传导至下游"
  - 预测: "中际旭创3天内上涨概率80%"
  - 时间敏感预警!
```

---

### 2. 混合推理引擎 (Hybrid Reasoning)

**核心问题**: 神经网络缺解释性，符号规则缺灵活性

**双轨架构**:
```python
class HybridReasoningEngine:
    def reason(query, context):
        # 轨道1: 符号层 (CSO使用)
        symbolic_result = symbolic_reasoning(query)
        # 如: "持仓>20% → REJECT"
        
        # 轨道2: 神经层 (UZI使用)
        neural_result = gnn_reasoning(query)
        # 如: "发现隐含关联CPO板块"
        
        # 动态仲裁
        if symbolic_result.action in ['REJECT']:
            return symbolic_result  # 硬约束优先
        else:
            return neural_result    # 神经推理为主
```

**使用场景**:
| 管理者 | 主要推理层 | 使用案例 |
|--------|-----------|----------|
| CSO | 符号层 | 合规检查、硬性风控规则 |
| UZI | 神经层 | 研报关联挖掘、隐含关系发现 |
| CIO | 混合层 | 投资决策，兼顾规则与机会 |

---

### 3. 图-文对齐 (Graph-Text Alignment)

**核心问题**: LLM与KG各自为政，信息孤岛

**双向融合**:
```python
# KG → LLM: 知识引导注意力
news = "英伟达发布B100，光模块需求激增"
kg_guided_attention(news, focus="英伟达")
→ 高权重词: ["英伟达", "光模块", "B100"]
→ 低权重词: ["天气", "体育", "娱乐"]

# LLM → KG: 事实提取与更新
facts = llm_extract_facts(news)
→ (NVIDIA, release, B100, confidence=0.95)
→ (CPO, demand_surge, True, confidence=0.85)

# 实时更新KG并触发GNN重算
update_kg_triples(facts)
trigger_gnn_recalculate()

# L1技能调用示例
technical_skill.query("MACD金叉关联板块?")
→ gnn_expand_neighbors("MACD", depth=2)
→ 返回: [CPO, 光通信, 中际旭创]
```
