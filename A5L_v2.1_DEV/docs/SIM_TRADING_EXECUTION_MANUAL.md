## 六、质量红线 (CSO必须执行)

### 6.1 数据校验 checklist

每日收盘后，CSO必须验证:

- [ ] 所有持仓标的价格是否来自真实数据源
- [ ] 今日交易价格是否在合理区间 (±20% vs 昨收)
- [ ] 盈亏计算是否正确 (市值 - 成本)
- [ ] 交易费用是否按真实费率计算
- [ ] 交易日历是否正确拦截非交易日

### 6.2 异常处理流程

```
发现异常 → 立即标记 → 通知Chief → 暂停交易 → 修复问题 → 验证恢复
```
