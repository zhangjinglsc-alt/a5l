# Unified Framework SKILL

## 描述

统一架构框架，整合技术分析层、数据获取层、AI应用层、验证指标层，提供完整的投资研究和交易支持体系。

## 使用方法

触发此 Skill 的指令：
- `统一框架` - 启动统一框架
- `系统架构` - 查看系统架构
- `框架集成` - 框架集成分析

## 四层架构

### 技术分析层
- 技术分析 → 股量分析
- 股量测试 → 因子测试
- 证券分析 → 比动分析
- 心理分析 → 情绪分析

### 数据获取层
- 统一股票价格 (unified-stock-price)
- AkShare数据 (aakshare)
- Web搜索 (coze-web-search, exa-web-search)
- 新闻API (unified-news-aggregator)
- 实时汇率 (fx-factor-monitor)
- 板块ETF (sector-etf-monitor)

### AI应用层
- AI大模型 (AI-LLM)
- AI应用 (AI-apps)
- AI+工业 (ai-manufacturing)
- 低空经济 (low-altitude)
- 新材料 (material)
- 存储 (storage)
- 液冷 (liquid-cooling)
- 具身智能 (embodied-ai)
- 测试测量 (test-measurement)

### 验证指标层
- 验证指标跟踪 (track_validation_metrics)
- 错误固化 (self-improving-agent)
- 用户纠正
- 成功案例

## 工作流程

```
数据获取 → 分析处理 → AI辅助 → 验证输出
```
