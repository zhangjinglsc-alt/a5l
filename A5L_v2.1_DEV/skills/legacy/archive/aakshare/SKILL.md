# AkShare 数据层 SKILL

## 描述

统一的股票数据接口，基于AkShare开源金融数据接口，自动多源切换，获取股票价格、基本面、市场数据、财务数据、技术指标、新闻数据，适用于股票数据查询、价格分析、财务分析、市场研究。

## 使用方法

触发此 Skill 的指令：
- `股票数据` - 获取A股数据
- `价格` - 查询实时价格
- `财务分析` - 获取财务数据
- `基本面` - 基本面分析
- `市场数据` - 市场数据查询

## 数据类型

### 股票基础数据
- 股票列表
- 实时行情
- 历史K线
- 分笔数据

### 财务数据
- 财务报表（资产负债表、利润表、现金流量表）
- 财务指标（ROE、毛利率、净利率等）
- 业绩报表
- 业绩预告

### 市场数据
- 大盘指数
- 行业板块
- 概念板块
- 资金流向

### 宏观数据
- GDP、CPI、PPI
- 货币供应量
- 利率数据
- 汇率数据

### 新闻数据
- 财经新闻
- 公告信息
- 研报数据

## 使用示例

```python
import akshare as ak

# 获取股票列表
stock_list = ak.stock_zh_a_spot()

# 获取历史数据
history = ak.stock_zh_a_hist(symbol="000001", period="daily")

# 获取财务报表
balance_sheet = ak.stock_balance_sheet_by_report_em(symbol="000001")
```
