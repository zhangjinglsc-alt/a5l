# Healthcheck SKILL

description: 主机安全与风险容忍度配置，系统安全增强，防火墙、SSH配置、安全检测、备份验证、加密存储、权限管理、端口管理、日志分析。

## 描述

主机安全与风险容忍度配置，系统安全增强，防火墙、SSH配置、安全检测、备份验证、加密存储、权限管理、端口管理、日志分析，适用于系统安全、主机安全、防火墙、SSH配置、安全检测、备份验证、权限管理、端口管理、日志分析。

## 使用方法

触发此 Skill 的指令：
- `系统安全` - 检查系统安全
- `主机安全` - 主机安全检测
- `防火墙` - 防火墙配置
- `SSH配置` - SSH安全配置
- `安全检测` - 安全漏洞检测
- `备份验证` - 验证备份完整性
- `权限管理` - 管理用户权限
- `端口管理` - 管理网络端口
- `日志分析` - 分析系统日志

## 安全检查项

### 系统安全
- 系统更新检查
- 漏洞扫描
- 恶意软件检测
- 入侵检测

### 网络安全
- 防火墙配置
- 端口开放检查
- 网络流量监控
- DDoS防护

### SSH安全
- 禁用root登录
- 使用密钥认证
- 修改默认端口
- 登录失败限制

### 数据安全
- 备份策略验证
- 加密存储检查
- 敏感数据保护
- 数据泄露检测
