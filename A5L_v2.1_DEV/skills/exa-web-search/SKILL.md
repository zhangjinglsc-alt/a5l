# Exa Web Search SKILL

## 描述

Exa Web搜索工具，基于AI的语义搜索引擎，提供高质量、相关性强的搜索结果，适用于深度研究、竞品分析、信息收集。

## 使用方法

触发此 Skill 的指令：
- `exa搜索` - 使用Exa搜索
- `web搜索` - 网页搜索
- `深度搜索` - 深度信息搜索

## 功能特点

### AI语义搜索
- 理解搜索意图
- 语义匹配而非关键词匹配
- 高质量结果排序

### 搜索类型
- 实时搜索
- 历史归档
- 相似内容查找
- 内容摘要

### 使用场景
- 竞品调研
- 技术调研
- 市场研究
- 论文查找

## 使用示例

```
exa_search("人形机器人最新进展")
exa_search("固态电池技术路线")
exa_search("AI芯片市场份额")
```
