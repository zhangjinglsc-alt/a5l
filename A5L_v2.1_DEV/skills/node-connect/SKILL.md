# Node Connect SKILL

description: 节点连接诊断工具，用于诊断OpenClaw节点连接和配对失败、二维码连接失败、本地Wi-Fi可用性检查、端口冲突检查、DNS问题排查、网络检测。

## 描述

节点连接诊断工具，用于诊断OpenClaw节点连接和配对失败、二维码连接失败、本地Wi-Fi可用性检查、端口冲突检查、DNS问题排查、网络检测，适用于节点连接、配对失败、网络检测、DNS问题排查、端口冲突、网络问题排查。

## 使用方法

触发此 Skill 的指令：
- `节点连接` - 诊断节点连接
- `配对失败` - 解决配对失败问题
- `网络检测` - 检测网络问题
- `DNS问题排查` - 排查DNS问题
- `端口冲突` - 检查端口冲突
- `网络问题排查` - 综合网络排查

## 诊断流程

### 1. 本地网络检查
- WiFi连接状态
- 本地IP地址
- 网关可达性
- DNS解析测试

### 2. 端口检查
- 常用端口占用
- 端口冲突检测
- 防火墙规则
- 端口转发配置

### 3. 配对流程检查
- 配对码有效性
- 令牌状态
- 授权状态
- 设备注册状态

### 4. 远程连接检查
- 公网IP可达性
- Tailscale状态
- 中继服务器连接
- 端到端连通性

## 常见问题

### QR码连接失败
- 检查网络连接
- 确认时间同步
- 验证配对码
- 检查防火墙

### 本地WiFi可用但远程不可用
- 检查端口转发
- 验证公网IP
- 检查Tailscale
- 检查中继配置

### 配对授权过期
- 重新生成配对码
- 更新授权令牌
- 重新配对设备
