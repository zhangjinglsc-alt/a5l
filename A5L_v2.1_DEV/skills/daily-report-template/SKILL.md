description: A5L持仓日报完整生成模板，包含A+H市场环境、4账户持仓归因、模拟盘分析、L3综合研判、风险评估、明日操作建议等全模块。

**版本**: v3.0  
**适用场景**: 每日17:30生成融合版持仓日报  
**SKILL调用**: 37个  
**核心特色**: 全SKILL支撑 + A5L深度思考 + 空方视角审查
