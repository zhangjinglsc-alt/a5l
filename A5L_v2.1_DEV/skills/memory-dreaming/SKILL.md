# Memory Dreaming SKILL

## 描述

记忆梦境管理，适用于梦境管理、创意挖掘、用户潜意识分析。

## 使用方法

触发此 Skill 的指令：
- `梦境` - 记录和分析梦境
- `梦` - 梦境相关内容
- `梦的管理` - 管理梦境记录
- `记忆` - 关联记忆分析
- `潜意识` - 潜意识分析

## 功能

### 梦境记录
- 记录梦境内容
- 标记梦境情绪
- 记录醒来时间
- 关联日常事件

### 梦境分析
- 梦境主题识别
- 符号解析
- 情绪分析
- 模式识别

### 创意挖掘
- 从梦境提取创意
- 联想扩展
- 灵感记录
- 创意孵化

### 潜意识分析
- 隐藏需求识别
- 情绪状态评估
- 压力源分析
- 内心冲突发现

## 使用示例

```
【梦境记录】
日期: YYYY-MM-DD
醒来时间: HH:MM
情绪: [积极/中性/焦虑]

梦境内容:
[详细描述梦境场景、人物、情节]

联想与思考:
[与现实的关联]

创意提取:
[从梦境获得的灵感]
```
