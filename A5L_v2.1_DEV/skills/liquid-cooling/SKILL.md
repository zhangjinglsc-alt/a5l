# 液冷 SKILL

## 描述

液冷技术产业分析工具，覆盖数据中心液冷、AI服务器液冷、浸没式液冷、冷板式液冷、液冷基础设施等，适用于液冷产业投资分析、行业研究、概念股筛选。

## 使用方法

触发此 Skill 的指令：
- `液冷` - 液冷技术分析
- `数据中心液冷` - 数据中心液冷分析
- `AI服务器液冷` - AI服务器液冷分析
- `浸没式液冷` - 浸没式液冷分析
- `冷板式液冷` - 冷板式液冷分析

## 覆盖领域

### 液冷技术类型
- **冷板式液冷** - 主流方案，改造难度低
- **浸没式液冷** - 散热效率高，改造成本高
- **喷淋式液冷** - 新兴技术

### 应用场景
- 超算中心
- AI训练集群
- 高密度数据中心
- 边缘计算节点

### 产业链
- 液冷板
- 冷却液
- 快换接头
- 冷却分配单元(CDU)
- 冷却塔/干冷器
- 温控系统

## 驱动因素

### 需求驱动
- AI算力爆发
- 功率密度提升
- PUE政策要求
- 散热效率需求

### 技术趋势
- 从风冷向液冷过渡
- 冷板向浸没演进
- 氟化液国产化
- 模块化设计
