### 2. 实时风控系统 (14KB)
**文件**: `risk_manager.py`

**核心功能**:
- 组合希腊字母 (Delta/Gamma/Theta/Vega/Rho)
- VaR计算 (95%/99%置信度)
- 压力测试 (5种情景)
- 实时熔断机制

**希腊字母**:
```
Delta: 0.9575 (价格敏感度)
Gamma: 0.0127 (二阶敏感度)
Theta: 0.0006 (时间衰减)
Vega:  0.1649 (波动率敏感度)
Rho:   0.0018 (利率敏感度)
```

**VaR结果**:
```
组合价值: ¥750,000
VaR(95%): ¥169,738 (22.6%)
VaR(99%): ¥240,006 (32.0%)
```

**压力测试**:
```
2008金融危机: -82.5% ❌ 无法承受
2020疫情冲击: -60.1% ❌ 无法承受
利率急升:     -37.2% ✅ 可承受
地缘冲突:     -29.7% ✅ 可承受
市场崩盘:     -99.9% ❌ 无法承受
```

### 3. 券商API模拟器 (16KB)

**核心功能**:
- A股券商API (100股整数倍限制)
- 美股券商API (无限制)
