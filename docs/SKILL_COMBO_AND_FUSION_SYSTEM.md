# SKILL组合技与融合训练系统 v1.0

## 一、核心概念

### 1.1 组合技 (Skill Combo)
**定义**: 两个或多个SKILL协同使用，产生1+1>2的效果
**特点**: 保持各SKILL独立性，按特定顺序/条件触发

### 1.2 融合技 (Skill Fusion)
**定义**: 多个SKILL深度整合，形成新的复合能力
**特点**: 产生全新的SKILL，具有独特的属性

### 1.3 协同效应 (Synergy)
**定义**: SKILL组合时产生的额外增益
**计算**: 总效果 = Σ(各SKILL效果) × 协同系数(1.2-2.0)

## 四、组合技训练方法

### 4.1 组合熟练度 (Combo Mastery)
**定义**: 特定组合技的协同效果发挥程度
**计算**: Combo_Mastery = avg(组成SKILL熟练度) × 协同系数 × 训练次数系数

**等级划分**:
- L1 初识: 0-30% (刚掌握组合)
- L2 熟练: 30-60% (能稳定触发)
- L3 精通: 60-80% (发挥80%效果)
- L4 大师: 80-95% (发挥95%效果)
- L5 传说: 95%+ (完美发挥+创新)

### 4.2 训练流程

#### 阶段1: 单SKILL达标
- 所有组成SKILL达到指定熟练度
- 各SKILL独立任务成功率>80%

#### 阶段2: 组合基础训练
- 按照固定顺序执行组合
- 反复练习触发条件识别
- 目标: 组合执行时间缩短50%

#### 阶段3: 协同优化
- 寻找最佳执行顺序
- 优化触发时机
- 目标: 协同系数提升至1.5+

#### 阶段4: 实战融合
- 真实场景应用
- 根据结果反馈调整
- 目标: 组合熟练度80%+

#### 阶段5: 创新演化 (高阶)
- 基于组合创造新用法
- 跨组合技串联
- 目标: 组合熟练度95%+
