## 🔧 学习机制详解

### 阶段1: 从飞书云文档学习

**目标SKILL**: 行业研究、催化剂分析、因子投资、价值投资等

**学习流程**:
```
扫描飞书文档目录
    ↓
匹配SKILL相关文档 (关键词匹配)
    ↓
提取核心知识 (投资逻辑、方法论、案例)
    ↓
结构化存储到知识库
    ↓
更新SKILL熟练度 (+0.1%~1% per item)
```

**示例**:
```
文档: "DFAU产业深度研究.md"
    ↓

  • 核心观点: "DFAU是CPO产业链价值皇冠"

  • industry_research +0.5%
```

## 📈 学习效果量化

### 熟练度提升算法

```python
新熟练度 = 旧熟练度 + 基础加成 + 知识加成 + 深度加成

其中:
  基础加成 = 0.001 (每次学习)
  知识加成 = min(知识项数 × 0.0001, 0.005)
  深度加成 = 学习深度 × 0.001

示例:
  industry_research 学习3项知识
  = 0.001 + (3 × 0.0001) + (3 × 0.001)
  = 0.001 + 0.0003 + 0.003
  = +0.43% 熟练度
```
