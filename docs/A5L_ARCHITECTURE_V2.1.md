## 🔄 反馈闭环详细流程

### 阶段1: 执行 (Execute)
```
CIO决策 → 调用Buffett策略 → 执行买入 → 记录执行
```

### 阶段2: 回写 (Write Back)
```
执行结果 ──► SkillFeedbackSystem.record_execution()
         ──► 创建KG实体: exec_buffett_v12_xxx
         ──► 关联: skill_buffett, stock_xxx, regime_熊市
```

### 阶段3: 计算 (Calculate)
```
SkillFeedbackSystem.calculate_skill_confidence('buffett_v12')
         ──► 胜率: 100% (5胜0负)
         ──► 平均收益: 15%
         ──► 最佳环境: 熊市
         ──► 综合置信度: 62/100
```

### 阶段4: 进化 (Evolve)
```
获取进化建议:
  → "该技能在熊市表现最佳"
  → "建议: 在熊市时增加该技能权重"
  → 动态加载器自动调整
```

### 阶段5: 应用 (Apply)
```
市场环境检测 → 识别为熊市
           → 动态加载Buffett策略 (权重50%)
           → 降低Momentum策略权重
           → 更优的决策结果
```

## 🎯 技能热插拔机制

### 市场环境识别
```python
# 基于多维度指标
trend_score = 60      # 趋势分数 (-100 to 100)
volatility = 0.15     # 波动率
volume_trend = 1.2    # 成交量趋势

regime = detect_market_regime(trend_score, volatility, volume_trend)
# 结果: MarketRegime.BULL (牛市)
```

### 技能组合动态生成
```python
# 根据市场环境筛选技能
suitable_skills = [
    skill for skill in skill_library
    if regime in skill.best_regimes
    and skill.confidence > 40
]

# 按置信度排序并分配权重
selected = suitable_skills[:4]  # Top 4
weights = normalize([s.confidence for s in selected])
```

### 运行时热插拔
```python
# 加载新技能
loader.hot_swap_skill('yangguan_v12', 'load')

# 卸载失效技能
loader.hot_swap_skill('momentum_v8', 'unload')

# 重新加载(更新参数)
loader.hot_swap_skill('buffett_v12', 'reload')
```
