## 🎯 SKILL小队快速调用指南

### 按场景调用

```python
# 场景1: 买入决策前分析
squad = form_squad([
    "@a5l/skill-catalyst-tier-framework",  # CIO-SQUAD-002
    "@a5l/skill-technical-analysis",        # CIO-SQUAD-001
    "@a5l/skill-bearish-perspective",       # CIO-SQUAD-001/CSO-SQUAD-001
    "@a5l/skill-private-banker-stock"       # PLATINUM-SQUAD-001
])

# 场景2: 每日复盘生成
squad = form_squad([
    "@a5l/skill-auto-briefing",             # COO-SQUAD-001
    "@a5l/skill-knowledge-guardian",        # KG-SQUAD-001
    "@a5l/skill-reading-analysis"           # RM-SQUAD-001
])

# 场景3: 产业深度研究
squad = form_squad([
    "@a5l/skill-industry-chain",            # PLATINUM-SQUAD-001
    "@a5l/skill-exa-web-search",            # KG-SQUAD-001
    "@a5l/skill-knowledge-graph"            # KG-SQUAD-001
])

# 场景4: 系统安全检查
squad = form_squad([
    "@a5l/skill-healthcheck",               # COO-SQUAD-002
    "@a5l/skill-data-integrity",            # CSO-SQUAD-001
    "@a5l/skill-guardrails-system"          # CSO-SQUAD-001
])
```

## 🔗 与Prime System集成

### Atom ID命名规范

```
@a5l/squad-{manager}-{squad_id}-{skill_name}

示例:
@a5l/squad-cio-001-factor-investing    # CIO小队1号-factor investing
@a5l/squad-platinum-001-private-banker # 白金小队1号-私行分析
```

### 小队关系定义

```json
{
  "id": "@a5l/squad-cio-001",
  "kind": "squad",
  "content": {
    "name": "投资决策小队",
    "manager": "@a5l/persona-cio",
    "members": [
      "@a5l/skill-factor-investing",
      "@a5l/skill-stock-five-steps",
      "..."
    ]
  },
  "edges": {
    "managed_by": ["@a5l/persona-cio"],
    "collaborates_with": ["@a5l/squad-platinum-001"],
    "requires": ["@a5l/squad-ca-001"]
  }
}
```
