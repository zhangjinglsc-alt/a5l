### 步骤3: 配置定时自动同步

#### 方式A: 系统定时任务 (推荐)

```bash
# 编辑定时任务
crontab -e

# 添加每日23:30自动同步
0 23 * * * cd /workspace/projects/workspace && python3 TOOLS/feishu_cloud_sync.py >> logs/sync.log 2>&1

# 保存退出
```

#### 方式B: OpenClaw内置Cron

```bash
# 添加每日同步任务到OpenClaw
openclaw cron add \
  --name "feishu-daily-sync" \
  --schedule "0 23 * * *" \
  --command "python3 TOOLS/feishu_cloud_sync.py"
```
