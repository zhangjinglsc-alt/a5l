## 自动同步策略

### 同步触发条件

| 触发条件 | 同步内容 | 频率 |
|----------|----------|------|
| 每日23:00 | 当日情景记忆 + 数据更新 | 每日 |
| 每周日23:00 | 周报 + 系统完整备份 | 每周 |
| Goal里程碑达成 | 目标更新 + 成果归档 | 实时 |
| SKILL更新 | 技能注册表更新 | 实时 |
| SOUL重大更新 | 人格宪章版本更新 | 实时 |

### 同步内容映射

| 本地文件 | 飞书文档路径 | 同步方式 |
|----------|--------------|----------|
| `SOUL.md` | `01-SOUL/SOUL-人格宪章.md` | 覆盖更新 |
| `SKILL_REGISTRY.json` | `02-SKILL/SKILL-REGISTRY-技能总表.md` | 转换为表格 |
| `MEMORY.md` | `03-MEMORY/MEMORY-长期记忆精华.md` | 追加更新 |
| `memory/YYYY-MM-DD.md` | `03-MEMORY/每日情景记忆/YYYY-MM-DD.md` | 新建 |
| `data/goals/*.json` | `04-GOAL/活跃目标/*.md` | 转换为文档 |
| `data/portfolio/*.json` | `06-DATA/DATA-持仓数据.md` | 转换为表格 |
| 生成的报告 | `07-REPORTS/分类归档/` | 新建 |

## 调用方式

### 1. 通过飞书搜索
在飞书搜索框输入关键词快速定位：
- "技能" → 找到SKILL注册表
- "2026-05-01" → 找到当天记忆
- "L3进化" → 找到目标进展
- "持仓" → 找到最新持仓数据

### 2. 通过飞书多维表格筛选
在SKILL表格中可按分类、熟练度筛选
在持仓表格中可按盈亏、账户筛选

### 3. 通过链接直接访问
每个文档都有唯一链接，可在对话中引用：
- "详见 [SOUL人格宪章](链接)"
- "参考 [2026-05-01情景记忆](链接)"
