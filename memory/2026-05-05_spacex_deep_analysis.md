3. **产业链全景**: 从宏观（太空经济）到中观（商业航天）到微观（细分环节）

5. **投资导向明确**: 最终落脚到可执行的投资建议

### 2.2 历史类比方法（"铁路时代"类比）

研报核心类比逻辑是**"商业航天=工业革命的铁路体系"**，具体类比维度：

| 类比维度 | 工业革命的铁路 | 当代商业航天 |
|----------|----------------|--------------|
| **技术本质** | 交通技术从工程能力转向商业化基础设施 | 航天运输从一次性任务转向可复用的基础设施 |
| **成本逻辑** | 蒸汽机效率突破→单位运输成本大幅下降（从0.48便士/吨英尺降至不足1美分） | 可回收火箭技术→LEO发射成本从5万美元/kg降至数千美元/kg |
| **经济影响** | 降低空间摩擦→整合全国市场、塑造新需求（跨区域分工、规模化工业） | 降低太空可达成本→扩展经济边界、催生深空经济（卫星互联网、太空制造） |
| **发展阶段** | 早期：低频次、高成本；后期：规模化、低成本 | 当前：从"高成本任务"转向"低成本基础设施"，进入规模化前夕 |
| **需求创造** | 铁路网络成型后，跨区域分工、规模化生产逐步出现 | 航天运输能力接近基础设施形态，需求随后释放 |

#### 类比的核心洞见

> "交通基础设施的需求是在成本下降、网络成型后被创造出来的"

这一历史类比打破了"商业航天需求不足"的质疑，证明：
- 铁路历史证明，交通基础设施在完成商业化并实现成本系统性下降之前，其潜在需求往往并不可见
- 商业航天在当前阶段所扮演的角色，并非对既有太空经济需求的简单满足，而更可能处于"交通条件先行、需求随后释放"的早期阶段

### 2.3 定量分析方法（成本效益测算、市场规模计算等）

研报使用多种定量工具强化论点：

#### 成本效益测算模型

**猎鹰9号复用成本测算**:
```
基础参数:
- 单次发射报价: ~7000万美元
- 边际运营成本: ~1500万美元
- 最高复用次数: 32次（截至2026年1月）

成本递减模型:
- 复用10次+时，单次平均成本稳定在~1700万美元
- 单次成本降幅趋近70%
- 一子级成本稀释至约94万美元
- 平均成本下探至1610万美元

毛利率:
- 2025年毛利率约79%
- 2026年预计达到约80%
```

**星舰成本测算**:
```
当前阶段:
- 研发与基础设施投入: ~50亿美元
- 单次硬件损耗: ~1亿美元
- 仍处于亏损特征

成熟运营期预测:
- 单次边际运营成本: ~200万美元
- 市场报价: ~2000万美元
- 单次发射毛利率: ~90%
- 稳态发射频次: 1000次/年
- 年度营收预测: ~200亿美元
- 年度毛利润预测: ~180亿美元
```

**太空算力中心经济性测算（Andrew McCalip模型）**:
```
当前条件（SpaceX V2 mini参数）:
- 发射成本: 500美元/kg
- 每瓦成本: 51.1美元/W
- 地面数据中心成本: 15.85美元/W
- 当前为地面成本的3倍以上

经济性成立条件:
- 发射成本: 200美元/kg
- 卫星硬件成本: 8美元/W
- 比功率: 100W/kg
- 每瓦成本: 16.02美元/W（与地面类似）

算力缺口测算（1TW目标）:
- 运力层面: 需要Starship V3实现7分钟/次发射频率
- 算力层面: 需要新增H100 14.29亿个/年，或B200 8.33亿个/年
```
