- **核心文件**: `ARCHITECT_5L/layer3_analysis/analyzers/bearish_perspective_analyzer.py` (24,546 bytes)
- **核心理念**: "多方观点让你看到机会，空方视角让你看到风险"
- **风险自查清单 (6大维度24项)**:
  | 维度 | 检查内容 | 示例 |
  |------|----------|------|
  | 估值风险 | PE/PB/PEG/历史分位 | PE是否高于行业50%? |
  | 技术风险 | 超买/背离/支撑/趋势 | RSI是否超过70? |
  | 基本面风险 | 业绩/负债/现金流/减持 | 净利润是否连续下滑? |
  | 市场风险 | 大盘/板块/VIX/政策 | 大盘是否跌破支撑? |
  | 流动性风险 | 成交额/换手率/解禁 | 成交额是否低于1亿? |
  | 集中度风险 | 客户/供应商/产品依赖 | 前五大客户占比>50%? |
- **策略逻辑审查 (6维度反驳)**:
  - 逻辑: 买入逻辑是否严谨 (避免假突破)
  - 数据: 业绩是否支撑估值 (避免估值泡沫)
  - 市场: 环境是否配合 (避免逆势操作)
  - 时机: 入场时机是否合适 (避免业绩雷)
  - 仓位: 风险控制是否到位 (避免过度集中)
  - 退出: 止损止盈是否明确 (避免无纪律)
- **风险评分系统 (0-100)**:
  ```
  🔴 70-100分: 极高风险 - 强烈建议观望
  🟠 50-69分: 高风险 - 谨慎参与
  🟡 30-49分: 中等风险 - 控制仓位
  🟢 10-29分: 低风险 - 可以参与
  ⚪ 0-9分: 安全 - 按计划执行
  ```
