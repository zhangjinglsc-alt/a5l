## 💡 核心能力矩阵

### 自修复自动化
```
错误发生
    ↓
自动分类 (严重程度/类型/难度/范围)
    ↓
策略匹配 (语法/逻辑/数据/网络/权限)
    ↓
沙箱预执行验证
    ↓
效果评估 (单元/集成/回归/性能)
    ↓
├─ 通过 → 自动应用 → 回写KG
└─ 失败 → 人机协同审批 → 人工处理
```

### 预测性维护
```
数据采集 (系统/应用/业务指标)
    ↓
健康预测 (LSTM+Prophet, 1h~7d)
    ↓
异常检测 (统计/ML/DL/GNN)
    ↓
预警分级 (P0/P1/P2/P3)
    ↓
├─ P0/P1 → 立即通知 + 自动修复
├─ P2 → 24h内关注 + 预防性维护
└─ P3 → 趋势观察 + 容量规划
```
