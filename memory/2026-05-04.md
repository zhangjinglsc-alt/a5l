### 1. Temporal KG-NN (时序图神经网络) ⏱️

**核心突破**: 从静态图到时序演化

**技术实现**:
- **历史编码**: R-GCN处理时间切片上的子图
- **演化单元**: GRU/LSTM更新演化嵌入 (Evolutionary Embedding)
- **动态预测**: 预测关系随时间的转移

**实战演示**:
```
源杰科技(光芯片)异动 +15%
↓
Temporal KG-NN分析历史模式:
  "上游异动平均3天后传导至下游"
↓
预测: "中际旭创3天内上涨概率80%"
↓
时间敏感预警!
```

**与传统KG对比**:
- 传统: 找到静态关联"中际旭创" (无时间概念)
- Temporal: 预测"3天后传导,概率80%" (时序敏感)

### 2. Hybrid Reasoning Engine (混合推理引擎) ⚖️

**核心突破**: 神经+符号双轨制

**架构设计**:
```
神经层 (GAT/GNN)          符号层 (逻辑规则)
├── 模糊匹配              ├── 硬约束
├── 关联发现              ├── 合规检查
├── UZI主要使用           ├── CSO主要使用
└── 发现"CPO关联"         └── "持仓>20%→REJECT"
           ↓                        ↓
        动态仲裁 (门控机制)
           ↓
    IF 符号触发硬约束 → 符号优先
    ELSE IF 时序敏感 → 时序推理
    ELSE → 神经为主+符号校验
```

**管理者分工**:
| 管理者 | 推理层 | 使用场景 |
|--------|--------|----------|
| CSO | 符号层 | 合规检查、硬性风控 |
| UZI | 神经层 | 研报关联、隐含关系 |
| CIO | 混合层 | 投资决策、规则+机会 |

### 3. Graph-Text Alignment (图-文对齐) 🔗

**核心突破**: LLM与KG实时双向互动

**双向融合**:
```
KG → LLM (知识引导注意力):
  新闻: "英伟达发布B100,光模块需求激增"
  KG引导: 关注"英伟达"→"光模块"→"CPO"
  注意力掩码: {"英伟达":1.0, "光模块":1.0, "其他":0.3}
  
LLM → KG (事实提取与更新):
  LLM提取: (NVIDIA, release, B100, 0.95)
  实时更新KG三元组
  触发GNN重新计算嵌入
  
GNN扩展查询 (L1技能调用):
  用户: "MACD金叉关联板块?"
  GNN: 扩展到"CPO"、"光通信"、"中际旭创"
```
