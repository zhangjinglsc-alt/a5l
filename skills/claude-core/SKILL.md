# Claude - 核心智能体 (Core Agent)

> **版本**: v1.0.0  
> **代号**: CLAW-Core  
> **定位**: A5L系统元控制层 - 统一自我诊断、修复、进化  
> **创建时间**: 2026-05-08

**描述**: Claude是A5L系统的核心智能体，统一整合Self-Diagnostics(诊断)、Self-Healing(修复)、Self-Evolution(进化)三大能力。作为系统的"中枢神经系统"，Claude持续监控系统健康，自动修复基础设施问题，并驱动能力进化。替代分散的多个引擎，提供统一的自我管理接口。

---

## 🎯 为什么要统一？

### 原有问题

| 问题 | 现象 | 解决方案 |
|:-----|:-----|:---------|
| **代码分散** | 3个引擎各自为政 | **统一核心**: Claude统一调度 |
| **职责模糊** | 诊断/修复/进化边界不清 | **分层架构**: 感知→决策→执行 |
| **重复触发** | 多个定时任务各自运行 | **统一心跳**: 单次触发全流程 |
| **难以扩展** | 新增能力要改多个文件 | **插件化**: 新能力即插即用 |

### 统一后的优势

```
Before:                    After:
├─ diagnostics.py          ├─ claude.py (统一核心)
├─ healing.py              │   ├─ 感知层 (检测)
├─ evolution.py            │   ├─ 认知层 (决策)
└─ (各自独立运行)          │   └─ 执行层 (行动)
                           └─ 统一调度，一次触发
```

```
│  └── 进化建议 (复杂问题提交给Chief)                          │
└─────────────────────────────────────────────────────────────┘
```

## 🔄 统一工作流

### 单次触发，全流程处理

```
Claude Heartbeat (每日03:00触发)
    ↓
【感知层】全面扫描
    ├── 基础设施检查
    ├── SKILL系统健康
    ├── 知识库完整性
    └── 预测性风险扫描
    ↓
【认知层】智能决策
    ├── 问题分类: 紧急/重要/轻微
    ├── 策略选择: 自动修复/建议人工/忽略
    └── 优先级排序
    ↓
【执行层】分级处理
    ├── P0紧急 → 立即自动修复
    ├── P1重要 → 自动修复 + 通知Chief
    ├── P2轻微 → 记录待办
    └── P3建议 → 生成进化提案
    ↓
【归档层】知识沉淀
    ├── 更新系统健康档案
    ├── 生成每日简报
    └── 同步到飞书 (如配置)
```

## 📋 核心能力

### 1. 自我诊断 (Self-Diagnostics)

**检测维度** (7个):
1. 核心文件完整性
2. SKILL系统健康
3. Git同步状态
4. 持仓数据一致性
5. 定时任务状态
6. 知识库完整性
7. 预测性风险扫描

**输出**: 系统健康评分 (0-100)

### 2. 自我修复 (Self-Healing)

**自动修复范围**:
- ✅ 创建缺失目录
- ✅ 修复Git同步 (自动提交)
- ✅ 补齐日志文件
- ✅ 同步配置更新
- ✅ 推送GitHub

**需人工介入**:
- 🔴 核心文件缺失
- 🔴 重大SKILL冲突
- 🔴 架构级重构

### 3. 自我进化 (Self-Evolution)

**进化触发条件**:
- 系统健康分 < 70 持续3天
- SKILL冲突检测阳性
- Chief主动提出需求

**进化提案生成**:
```markdown
## 进化提案 #2026-05-008

**问题**: SKILL数量不足 (当前59，目标60+)
**建议**: 创建"量化交易"SKILL
**优先级**: P2
**预计工作量**: 2小时
**价值**: 填补策略执行层缺口
```

## 🚀 使用方式

### 命令行接口

```bash
# 手动触发完整检查
python3 -m claude check

# 仅诊断
python3 -m claude diagnose

# 仅修复
python3 -m claude heal

# 生成进化提案
python3 -m claude evolve

# 查看健康档案
python3 -m claude status
```

### Python API

```python
from claude import CoreAgent

claude = CoreAgent()

# 单次完整检查
report = claude.run_full_cycle()

# 仅诊断
diagnosis = claude.diagnose()

# 执行修复
fixes = claude.heal(diagnosis.issues)

# 生成进化建议
proposals = claude.suggest_evolution()
```
