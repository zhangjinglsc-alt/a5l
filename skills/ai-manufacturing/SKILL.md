# AI+工业制造 SKILL

## 描述

AI+工业制造领域分析工具，覆盖智能制造、工业4.0、机器视觉、预测性维护、数字孪生等，适用于工业AI投资分析、行业研究、概念股筛选。

## 使用方法

触发此 Skill 的指令：
- `AI工业` - AI+工业制造分析
- `智能制造` - 智能制造领域分析
- `工业4.0` - 工业4.0趋势分析
- `机器视觉` - 机器视觉产业链分析
