# Stoic Wealth Philosophy SKILL

## 描述

斯多葛财富哲学，财务管家视角的投资分析工具，用于个人财务跟踪、预算规划、现金流管理、投资组合管理、保险规划、税务优化、债务管理，适用于个人财务管理、家庭财务规划、投资组合管理、保险规划、税务优化、债务管理。

## 使用方法

触发此 Skill 的指令：
- `财务` - 财务管理
- `预算` - 预算规划
- `投资组合` - 投资组合管理
- `保险规划` - 保险规划
- `税务` - 税务优化
- `债务` - 债务管理

## 斯多葛原则

### 1. 控制二分法
- **可控**: 储蓄率、支出决策、投资策略
- **不可控**: 市场波动、利率变化、经济周期
- **专注**: 把精力放在可控的事情上

### 2. 逆境训练
- 预设最坏情况
- 建立财务缓冲
- 心理预备
- 危机应对

### 3. 简朴生活
- 区分需要与想要
- 控制生活方式膨胀
- 提高储蓄率
- 财务自由

### 4. 长期视角
- 时间复利
- 延迟满足
- 代际财富
- 遗产规划
