# Self-Evolution Core SKILL

## 描述

进化核心，错误记录+用户纠正+成功案例固化+智能更新+仪表盘+生成周报，适用于错误修复、用户纠正、成功模式固化、知识学习。

> **版本说明**: v2.0 - 当前主版本自我改进技能
> **替代技能**: `self-improving-agent` v1.x (已归档至 legacy/archive/)

## 使用方法

触发此 Skill 的指令：
- `自我进化` - 启动自我进化
- `错误修复` - 修复系统错误
- `用户纠正` - 学习用户纠正
- `成功模式固化` - 固化成功模式
- `知识学习` - 学习新知识

## 核心功能

### 1. 错误记录
- 自动错误捕获
- 错误分类标签
- 上下文记录
- 影响评估

### 2. 用户纠正学习
- 记录用户反馈
- 提取纠正要点
- 更新知识库
- 预防重复错误

### 3. 成功案例固化
- 识别成功模式
- 记录成功要素
- 更新最佳实践
- 推广成功经验

### 4. 智能更新
- 检测改进机会
- 生成更新方案
- 自动测试验证
- 版本发布

### 5. 仪表盘
- 系统健康度
- 错误趋势图
- 学习效率指标
- 进化进度追踪

### 6. 周报生成
- 本周错误统计
- 学习成果汇总
- 改进建议
- 下周计划
