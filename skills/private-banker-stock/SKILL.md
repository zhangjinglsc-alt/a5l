# Private Banker Stock SKILL

## 描述

私人投行股票分析工具，深度分析个股价值，提供专业投资建议，适用于高净值客户股票分析、价值投资、深度研究。

## 使用方法

触发此 Skill 的指令：
- `私人投行` - 启动私人投行分析
- `深度分析` - 深度股票分析
- `专业建议` - 获取专业投资建议

## 分析框架

### 1. 宏观环境
- 经济周期定位
- 货币政策环境
- 行业政策导向
- 国际形势影响

### 2. 行业分析
- 行业生命周期
- 竞争格局分析
- 产业链地位
- 进入壁垒评估

### 3. 公司研究
- 商业模式分析
- 竞争优势评估
- 管理团队评价
- 公司治理结构

### 4. 财务分析
- 盈利能力分析
- 成长性分析
- 运营效率分析
- 财务安全分析

### 5. 估值分析
- 绝对估值 (DCF)
- 相对估值 (PE/PB/PS)
- EV/EBITDA分析
- SOTP估值

### 6. 风险评估
- 经营风险
- 财务风险
- 政策风险
- 市场风险
