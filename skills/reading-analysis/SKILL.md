# Reading Analysis SKILL

## 描述

三重阅读法（精读+跳读+深读），适用于书籍阅读、研报速读、文献分析、深度思考。

## 使用方法

触发此 Skill 的指令：
- `读书` - 开始阅读分析
- `书籍` - 分析书籍内容
- `三重阅读法` - 应用三重阅读法
- `研报速读` - 快速阅读研报
- `文献分析` - 分析学术文献
- `深度思考` - 深度思考分析

## 三重阅读法

### 第一遍：跳读 (Skimming)
**目标**: 把握全局，了解框架
- 阅读目录、前言、结论
- 浏览章节标题和小标题
- 看图表、摘要、关键段落
- 标记重点章节

### 第二遍：精读 (Reading)
**目标**: 理解细节，掌握内容
- 逐章细读核心内容
- 做笔记、划重点
- 理解论证逻辑
- 记录疑问和思考

### 第三遍：深读 (Deep Reading)
**目标**: 批判思考，内化知识
- 反思作者观点
- 联系已有知识
- 评估论证质量
- 形成自己的观点
- 实践应用思考

## 应用场景

### 书籍阅读
- 先跳读把握全书结构
- 再精读理解核心思想
- 最后深读形成自己的理解

### 研报速读
- 5分钟跳读：结论、评级、目标价
- 15分钟精读：核心逻辑、关键数据
- 30分钟深读：假设条件、风险提示

### 文献分析
- 跳读：摘要、引言、结论
- 精读：方法、结果、讨论
- 深读：研究局限、未来方向
