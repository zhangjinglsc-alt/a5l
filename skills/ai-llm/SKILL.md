# AI-LLM SKILL

## 描述

AI大模型综合工具，覆盖GPT、Claude、Gemini、Llama等主流大模型，支持模型选择、Prompt工程、多模态交互、Agent构建等。

## 使用方法

触发此 Skill 的指令：
- `AI大模型` - AI大模型工具
- `LLM` - 大语言模型
- `模型选择` - 选择合适的AI模型
- `Prompt工程` - Prompt优化

## 支持的模型

### 闭源模型
- **GPT-4/GPT-4o** - OpenAI旗舰模型
- **Claude 3** - Anthropic模型
- **Gemini** - Google模型
- **Kimi** - Moonshot模型
- **文心一言** - 百度模型

### 开源模型
- **Llama 3** - Meta模型
- **Qwen** - 阿里模型
- **ChatGLM** - 清华模型
- **Baichuan** - 百川模型
