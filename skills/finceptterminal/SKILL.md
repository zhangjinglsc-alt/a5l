# FinceptTerminal Skill

## 名称
finceptterminal - 金融数据终端解析器

## 描述
专业金融数据文件解析工具，支持多种历史数据格式解析、清洗和标准化，专为A5L CIO觉醒系统设计。

## 能力
- 解析CSV/JSON/Parquet格式的股票历史数据
- 支持日K/周K/月K数据标准化
- 数据质量检查和清洗
- 自动识别股票代码和日期格式
- 生成SQLite数据库导入格式
- 支持800MB+大文件流式处理

## 使用场景
1. 解析开盘啦历史数据zip文件
2. 标准化日K线数据格式
3. 数据质量验证和清洗
4. 生成ML训练数据集

## 输入
- 历史数据文件路径 (CSV/JSON/zip)
- 数据格式类型
- 输出目标 (SQLite/CSV)

## 输出
- 标准化的OHLCV数据
- 数据质量报告
- SQLite数据库文件
- 处理日志
