# 巴菲特价值投资思维 SKILL

## 描述

价值投资顶层思考框架：第一性原理+叙事冲击vs主线逻辑区分、三阶段交易框架+分歧-一致决策矩阵、市场偏好周期、供给冲击分析、地缘冲突交易节奏，适用于价值投资、择时策略、长期持有。

## 使用方法

触发此 Skill 的指令：
- `/巴菲特` - 启动巴菲特价值投资框架
- `/价值投资` - 价值投资分析
- `/长期持有` - 长期投资策略

## 核心思维框架

### 1. 第一性原理 (First Principles)
- **本质思考** - 剥离表象看本质
- **底层逻辑** - 商业本质与盈利逻辑
- **长期视角** - 10年以上的时间维度
- **能力圈** - 只投自己能理解的生意

### 2. 叙事冲击 vs 主线逻辑
- **叙事冲击** - 短期市场情绪驱动的波动
- **主线逻辑** - 公司长期价值的核心驱动
- **区分方法** - 判断是否影响未来现金流
- **应对策略** - 利用叙事冲击，坚守主线逻辑

### 3. 三阶段交易框架
- **第一阶段：分歧期** - 市场认知分歧，价值发现阶段
- **第二阶段：共识期** - 逻辑被验证，估值修复阶段
- **第三阶段：泡沫期** - 过度乐观，考虑离场阶段

### 4. 分歧-一致决策矩阵
| 市场状态 | 投资策略 | 仓位建议 |
|---------|---------|---------|
| 分歧-低估 | 大胆买入 | 重仓 |
| 共识-合理 | 持有观察 | 中等 |
| 一致-高估 | 逐步减仓 | 轻仓 |
| 分歧-迷茫 | 谨慎观望 | 轻仓/空仓 |

### 5. 市场偏好周期
- **价值风格周期** - 低估值因子表现期
- **成长风格周期** - 高成长因子表现期
- **周期切换信号** - 利率、通胀、盈利预期变化

### 6. 供给冲击分析
- **产能周期** - 行业供给变化对价格的影响
- **库存周期** - 补库存/去库存阶段判断
- **供给侧出清** - 寻找行业底部的机会

### 7. 地缘冲突交易节奏
- **冲突爆发期** - 避险情绪，避险资产上涨
- **僵持消耗期** - 市场适应，回归基本面
- **缓和/解决期** - 风险溢价回落，风险资产修复

## 巴菲特投资原则

1. **能力圈原则** - 只投资自己能理解的生意
2. **安全边际** - 以显著低于内在价值的价格买入
3. **护城河理论** - 寻找具有持久竞争优势的企业
4. **长期持有** - 最好的投资是永久持有
5. **逆向投资** - 别人贪婪我恐惧，别人恐惧我贪婪

## 分析流程

1. **商业模式分析** - 理解企业如何赚钱
2. **护城河评估** - 判断竞争优势可持续性
3. **财务分析** - ROE、毛利率、现金流质量
4. **管理层评估** - 诚信、能力、资本配置
5. **估值分析** - 确定合理买入价格
6. **买入决策** - 等待安全边际出现

## 输出

- 价值投资分析报告
- 买入/持有/卖出建议
- 目标价位与安全边际
- 长期跟踪要点
