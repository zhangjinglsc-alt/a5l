# 股票五步法分析 SKILL

## 描述

五步法分析框架（好公司+好未来+好价格+好买卖+好评级），适用于个股深度分析，进行个股深度挖掘和投资决策。

## 使用方法

触发此 Skill 的指令：
- `/个股分析` - 启动个股五步法分析
- `/五步法` - 使用五步法框架
- `/分析` - 开始深度分析

## 五步分析框架

### 第一步：好公司 (Good Company)
- **商业模式** - 盈利模式是否清晰可持续
- **竞争壁垒** - 护城河深度与宽度
- **管理层** - 管理团队能力与诚信
- **财务健康** - 资产负债表、现金流质量
- **行业地位** - 市场份额、议价能力

### 第二步：好未来 (Good Future)
- **行业前景** - 行业发展空间与趋势
- **成长驱动** - 收入增长的核心驱动力
- **政策环境** - 监管政策对行业的影响
- **技术变革** - 技术创新带来的机遇/威胁
- **竞争格局** - 未来竞争态势预判

### 第三步：好价格 (Good Price)
- **估值水平** - PE/PB/PS/PEG 历史分位
- **DCF估值** - 现金流折现内在价值
- **同业比较** - 行业相对估值
- **安全边际** - 价格与价值的差距
- **催化剂** - 潜在的估值提升因素

### 第四步：好买卖 (Good Timing)
- **买入时机** - 技术/基本面买点识别
- **仓位管理** - 分批建仓策略
- **止损设置** - 风险控制底线
- **止盈策略** - 获利了结方案
- **持有周期** - 投资时间框架

### 第五步：好评级 (Good Rating)
- **综合评分** - 多维度量化评分
- **投资建议** - 买入/持有/卖出评级
- **目标价位** - 合理价格区间
- **风险等级** - 投资风险评估
- **跟踪要点** - 后续关注指标
