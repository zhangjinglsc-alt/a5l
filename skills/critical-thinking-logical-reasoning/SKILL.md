# Critical Thinking & Logical Reasoning SKILL

## 描述

批判性思维与逻辑推理增强器，适用于批判性思维、逻辑推理、论证分析。

## 使用方法

触发此 Skill 的指令：
- `批判性思维` - 启动批判性思维分析
- `逻辑推理` - 逻辑推理分析
- `论证分析` - 分析论证结构

## 核心框架

### 1. 论证分析
- **论点识别** - 找出核心主张
- **论据评估** - 检验证据质量
- **逻辑结构** - 分析推理过程
- **隐含假设** - 挖掘未明说的前提

### 2. 谬误识别
- 人身攻击
- 稻草人谬误
- 滑坡谬误
- 假两难推理
- 诉诸权威
- 因果混淆

### 3. 证据评估
- 来源可靠性
- 数据质量
- 样本代表性
- 相关性vs因果性

### 4. 多角度思考
- 换位思考
- 反面论证
- 替代解释
- 边缘案例分析

## 应用方法

```
1. 明确问题/论点
2. 收集相关信息
3. 分析论证结构
4. 评估证据质量
5. 识别潜在谬误
6. 形成独立判断
7. 考虑替代观点
```
