# 新材料 SKILL

## 描述

新材料领域分析工具，覆盖碳纤维、石墨烯、高温合金、半导体材料、锂电材料、稀土功能材料等，适用于新材料投资分析、行业研究、概念股筛选。

## 使用方法

触发此 Skill 的指令：
- `新材料` - 新材料领域分析
- `碳纤维` - 碳纤维产业链分析
- `石墨烯` - 石墨烯应用分析
- `高温合金` - 高温合金分析
- `半导体材料` - 半导体材料分析
- `锂电材料` - 锂电材料分析
