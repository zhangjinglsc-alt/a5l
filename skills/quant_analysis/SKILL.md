# 量化分析 SKILL

## 描述

量化投资综合工具，包含财务分析、技术指标、风险评估、情绪分析、回测引擎、组合风险、策略回测，适用于量化策略构建、股票分析、投资研究。

## 使用方法

触发此 Skill 的指令：
- `/量化分析` - 启动量化分析
- `/回测` - 策略回测
- `/策略回测` - 详细策略回测分析
- `/股票分析` - 量化股票分析

## 功能模块

### 1. 财务分析
- **基本面筛选** - ROE、毛利率、营收增长
- **财务质量** - 现金流、应收账款、存货
- **估值分析** - PE、PB、PS、EV/EBITDA
- **杜邦分析** - ROE分解分析

### 2. 技术指标
- **趋势指标** - MA、MACD、Bollinger Bands
- **动量指标** - RSI、KDJ、Williams %R
- **成交量** - OBV、Volume MA、量价分析
- **波动率** - ATR、布林带宽度、历史波动率

### 3. 风险评估
- **风险指标** - 波动率、VaR、最大回撤
- **风险调整收益** - 夏普比率、索提诺比率、卡玛比率
- **尾部风险** - 偏度、峰度、下行风险
- **压力测试** - 极端市场环境测试

### 4. 情绪分析
- **市场情绪** - 恐慌/贪婪指数、VIX
- **资金流向** - 北向资金、主力资金、融资融券
- **舆情分析** - 新闻情绪、社交媒体情绪
- **机构行为** - 基金仓位、调研活动

### 5. 回测引擎
- **策略回测** - 历史数据策略验证
- **参数优化** - 网格搜索、遗传算法优化
- **过拟合检测** - 样本外测试、前向验证
- **绩效归因** - 收益来源分解

### 6. 组合风险
- **相关性分析** - 资产间相关系数矩阵
- **风险贡献** - 边际风险贡献度
- **风险预算** - 目标风险分配
- **情景分析** - 不同市场情景下的组合表现
