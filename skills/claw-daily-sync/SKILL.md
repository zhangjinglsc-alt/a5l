# CLAW Daily Sync SKILL

## 描述

每日4个自动检查点（06:30/12:30/15:00/18:00），双保险备份（Git本地+飞书），持仓数据自动同步，研报数据+日志+文献自动归档，适用于数据备份、同步、检查状态、数据完整性保障、备份检查。

## 使用方法

触发此 Skill 的指令：
- `数据备份` - 执行数据备份
- `同步` - 执行数据同步
- `检查状态` - 检查系统状态
- `数据完整性保障` - 保障数据完整性
- `备份检查` - 检查备份状态

## 每日检查点

### 06:30 - 早盘准备
- 数据备份
- 新闻聚合
- 持仓数据同步

### 12:30 - 午间复盘
- 上午交易数据备份
- 研报更新同步
- 日志归档

### 15:00 - 收盘数据
- 收盘数据获取
- 持仓盈亏计算
- 数据同步

### 18:00 - 日终归档
- 全量数据备份
- 飞书同步
- 完整性检查
- 日志归档

## 双保险备份

### 本地Git备份
- 版本控制
- 变更追踪
- 快速恢复

### 飞书云端同步
- 云端备份
- 多端访问
- 团队共享

## 自动归档

- 研报数据归档
- 交易日志归档
- 系统日志归档
- 文献资料归档
