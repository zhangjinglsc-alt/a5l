# 统一回测引擎 SKILL

## 描述

统一回测引擎，支持多策略（双均线/RSI/MACD/突破）、多数据源（AkShare/TQ/CSV）、回测报告、实盘模拟，适用于回测、策略回测、策略优化。

## 使用方法

触发此 Skill 的指令：
- `回测` - 启动回测
- `回测引擎` - 详细回测分析
- `策略回测` - 指定策略回测

## 支持的策略

### 1. 双均线策略
- 金叉买入，死叉卖出
- 可配置均线周期

### 2. RSI策略
- 超卖买入（RSI<30）
- 超买卖出（RSI>70）

### 3. MACD策略
- DIF上穿DEA买入
- DIF下穿DEA卖出

### 4. 突破策略
- 价格突破N日高点买入
- 价格跌破N日低点卖出

## 回测参数

- **初始资金** - 默认100万
- **回测周期** - 起始/结束日期
- **交易费用** - 佣金+印花税
- **滑点** - 成交价格滑点
- **仓位管理** - 固定金额/固定比例
