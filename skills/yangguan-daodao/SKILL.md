# 阳关大道超短线交易系统 SKILL

## 描述

超短线交易系统，核心逻辑：胜率不重要，盈亏比才重要。包含双均线策略（金叉、死叉）、RSI/ATR策略、MACD策略，适用于超短线、题材轮动、AI服务器核心票。

## 使用方法

触发此 Skill 的指令：
- `/阳关` - 启动阳关大道交易系统
- `/超短` - 超短线交易分析
- `/超短线` - 超短线操作策略

## 核心逻辑

### 盈亏比至上
- **胜率不重要** - 40%胜率也能盈利
- **盈亏比关键** - 追求 1:3 以上的盈亏比
- **截断亏损** - 严格止损，单笔亏损控制在2-3%
- **让利润奔跑** - 盈利头寸持有，追求大波段

## 交易策略

### 1. 双均线策略
- **金叉买入** - 短期均线上穿长期均线
- **死叉卖出** - 短期均线下穿长期均线
- **参数设置** - 5日/20日，10日/30日组合
- **适用场景** - 趋势明确的强势票

### 2. RSI策略
- **超卖买入** - RSI < 30，结合形态买入
- **超买卖出** - RSI > 70，考虑获利了结
- **背离信号** - 价格与RSI背离，趋势反转预警
- **参数设置** - 默认14日RSI

### 3. ATR策略
- **波动率过滤** - ATR放大时谨慎追高
- **止损设置** - 以2倍ATR作为止损距离
- **仓位管理** - 根据ATR调整仓位大小

### 4. MACD策略
- **金叉买入** - DIF上穿DEA，柱状线翻红
- **死叉卖出** - DIF下穿DEA，柱状线翻绿
- **零轴判断** - 零轴上方多头，零轴下方空头

## 交易纪律

### 入场规则
1. 趋势方向明确
2. 出现技术买点信号
3. 成交量配合放大
4. 板块/题材热度支持

### 出场规则
1. **止损出场** - 跌破止损位无条件离场
2. **止盈出场** - 达到目标位分批止盈
3. **技术出场** - 出现反转信号离场
4. **时间出场** - 持股超期未达预期离场

### 仓位管理
- **单票仓位** - 不超过总资金20%
- **总仓位** - 根据市场环境调整30-80%
- **加仓原则** - 盈利后才考虑加仓
- **减仓原则** - 亏损绝不补仓

## 交易流程

1. **盘前准备** - 筛选目标股票池
2. **开盘观察** - 观察竞价和开盘走势
3. **信号确认** - 等待技术信号确认
4. **执行交易** - 按计划入场
5. **持仓管理** - 跟踪止盈止损
6. **复盘总结** - 盘后总结得失

## 风险控制

- **单笔止损** - 不超过本金的2%
- **单日止损** - 不超过本金的5%
- **连续亏损** - 3次连续亏损后暂停交易
- **情绪波动** - 情绪不稳定时不交易

## 适用标的

- AI服务器核心票
- 题材龙头股
- 高波动率强势股
- 板块轮动领涨股

## 注意事项

- 超短线对执行力要求极高
- 严格止损是生存前提
- 避免频繁交易导致磨损
- 保持交易日志记录
