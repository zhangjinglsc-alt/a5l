# 财务计算器 SKILL

## 描述

财务计算器，包括复利计算、贴现计算、抵押贷款计算、分期贷款计算、折旧计算、汇率转换、债券定价、年金计算、ROI计算、IRR计算、NPV、贷款摊还计划，适用于财务计算、贷款分析、投资决策、风险评估。

## 使用方法

触发此 Skill 的指令：
- `财务计算` - 启动财务计算
- `财务分析` - 财务指标分析
- `贷款分析` - 贷款相关计算
- `投资决策` - 投资分析计算

## 计算功能

### 1. 复利计算
- 未来值 (FV) 计算
- 现值 (PV) 计算
- 利率计算
- 期数计算

### 2. 贴现计算
- 现金流贴现
- 永续年金现值
- 增长年金现值

### 3. 贷款计算
- 月供计算 (PMT)
- 剩余本金计算
- 贷款摊还表生成
- 提前还款节省计算

### 4. 折旧计算
- 直线法折旧
- 双倍余额递减法
- 年数总和法

### 5. 投资分析
- ROI (投资回报率)
- IRR (内部收益率)
- NPV (净现值)
- PI (盈利指数)

### 6. 债券计算
- 债券定价
- 到期收益率 (YTM)
- 久期计算

### 7. 汇率转换
- 实时汇率查询
- 历史汇率查询
- 多币种转换

## 使用示例

```
复利计算:
本金: 100,000
年利率: 8%
期限: 10年
复利次数: 每年1次

未来值 = 215,892

---

房贷计算:
贷款金额: 1,000,000
年利率: 4.2%
期限: 30年

月供 = 4,890
总利息 = 760,400
```
