# Agent 自我进化 SKILL

## 描述

自我进化系统，自动记录错误、学习解决方案、更新Skill，适用于错误修复、用户纠正、成功模式固化、知识学习。

## 使用方法

触发此 Skill 的指令：
- `错误修复` - 记录和修复错误
- `用户纠正` - 学习用户纠正
- `成功模式固化` - 固化成功经验
- `知识学习` - 学习新知识

## 功能模块

### 1. 错误记录
- 自动捕获错误
- 错误分类和标签
- 上下文信息记录
- 用户纠正关联

### 2. 解决方案学习
- 从错误中学习
- 记录修复方法
- 建立知识库
- 模式识别

### 3. Skill更新
- 自动检测改进点
- 生成Skill更新建议
- 版本控制
- 回滚机制

### 4. 成功模式固化
- 识别成功模式
- 记录最佳实践
- 更新Skill文档
- 知识传承

## 进化流程

```
错误发生 → 捕获记录 → 分析原因 → 学习方案 → 更新Skill → 验证效果
```

## 输出

- 错误日志
- 学习报告
- Skill更新建议
- 进化周报
