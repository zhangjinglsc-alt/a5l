# Sector ETF Monitor SKILL

## 描述

板块ETF监控工具，实时追踪行业板块ETF表现，分析板块轮动、资金流向、技术指标，适用于板块分析、行业配置、轮动策略。

## 使用方法

触发此 Skill 的指令：
- `板块ETF` - 板块ETF监控
- `行业板块` - 行业板块分析
- `板块轮动` - 板块轮动分析
- `资金流向` - 资金流向分析
