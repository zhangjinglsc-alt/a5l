# 研报数据完整性保障 SKILL

## 描述

研报数据完整性保障系统，每日备份+飞书同步+完整性复检，适用于研报数据管理、飞书同步、数据完整性、数据备份。

## 使用方法

触发此 Skill 的指令：
- `研报数据管理` - 管理研报数据
- `飞书同步` - 同步到飞书
- `数据完整性` - 检查数据完整性
- `数据备份` - 执行数据备份

## 功能

### 1. 数据备份
- 每日自动备份
- 本地Git版本控制
- 云端飞书同步
- 增量备份策略

### 2. 完整性检查
- 文件完整性校验
- 数据一致性检查
- 缺失数据检测
- 异常数据告警

### 3. 飞书同步
- 自动上传到飞书云空间
- 文件夹结构同步
- 版本历史管理
- 访问权限控制

### 4. 恢复机制
- 从备份恢复
- 版本回滚
- 数据修复
- 灾难恢复

## 备份策略

```
每日 06:30 - 本地Git备份
每日 12:30 - 飞书云端同步
每日 18:00 - 完整性复检
每周日 - 全量备份归档
```
