# Beancount 记账 SKILL

## 描述

专业个人财务记账工具，支持记账、账户管理、财务分析、投资跟踪、预算管理，适用于个人记账、财务分析、投资跟踪、预算管理。

## 使用方法

触发此 Skill 的指令：
- `记账` - 记录交易
- `财务分析` - 分析财务状况
- `投资跟踪` - 跟踪投资表现
- `预算管理` - 管理预算

## 功能

### 1. 复式记账
- 基于 Beancount 语法
- 资产 = 负债 + 权益
- 每笔交易借贷平衡

### 2. 账户管理
- 资产账户（现金、银行存款、投资）
- 负债账户（信用卡、贷款）
- 收入账户（工资、投资收益）
- 支出账户（餐饮、交通、购物）

### 3. 财务分析
- 收支报表
- 资产负债表
- 现金流量表
- 预算执行分析

### 4. 投资跟踪
- 持仓成本跟踪
- 已实现/未实现盈亏
- 投资回报率计算
- 资产配置分析

### 5. 预算管理
- 月度预算设置
- 分类预算分配
- 预算执行监控
- 超支预警

## 记账示例

```beancount
; 工资收入
2024-01-15 * "工资收入"
  Income:Salary:Company  -20,000 CNY
  Assets:Bank:Checking

; 日常支出
2024-01-16 * "超市购物" "Grocery"
  Expenses:Food:Grocery  350 CNY
  Assets:Bank:Checking

; 投资交易
2024-01-17 * "买入茅台"
  Assets:Investment:Stock:茅台  100 {1200 CNY}
  Assets:Bank:Checking  -120,000 CNY
```

## 报表输出

- 收支明细表
- 分类统计表
- 资产负债表
- 投资收益表
- 预算执行表
