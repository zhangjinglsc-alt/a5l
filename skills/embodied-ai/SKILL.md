# 具身智能 SKILL

## 描述

具身智能（Embodied AI）领域分析工具，覆盖人形机器人、自动驾驶、智能机器人、多模态AI、仿真训练等，适用于具身智能投资分析、行业研究、概念股筛选。

## 使用方法

触发此 Skill 的指令：
- `具身智能` - 具身智能分析
- `人形机器人` - 人形机器人分析
- `自动驾驶` - 自动驾驶分析
- `智能机器人` - 智能机器人分析
- `多模态AI` - 多模态AI分析
