# AI Apps SKILL

## 描述

AI应用综合工具，覆盖AI写作、AI绘画、AI视频、AI音频、AI编程等热门AI应用，适用于AI应用推荐、使用指南、效果对比。

## 使用方法

触发此 Skill 的指令：
- `AI应用` - AI应用工具
- `AI写作` - AI写作工具
- `AI绘画` - AI绘画工具
- `AI视频` - AI视频工具
- `AI编程` - AI编程工具

## 应用分类

### AI写作
- ChatGPT / Claude - 通用写作
- Jasper / Copy.ai - 营销文案
- Notion AI - 笔记助手
- 秘塔写作猫 - 中文写作

### AI绘画
- Midjourney - 艺术创作
- DALL-E 3 - OpenAI绘画
- Stable Diffusion - 开源绘画
- 文心一格 - 百度绘画

### AI视频
- Runway - AI视频编辑
- Pika - 视频生成
- HeyGen - 数字人
- Sora - OpenAI视频

### AI音频
- ElevenLabs - AI语音
- Suno - AI音乐
- Murf - 语音合成

### AI编程
- GitHub Copilot - 代码补全
- Cursor - AI编辑器
- Codeium - 免费替代
- Devin - AI程序员

## 使用建议

- 根据需求选择合适的工具
- 对比不同工具的效果
- 学习Prompt技巧
- 关注价格和限制
