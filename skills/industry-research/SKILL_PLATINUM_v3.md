## 一、白金分析师的六大核心能力

### 1.1 找到并监控关键因素（2-4点原则）

**核心原则**: 每支股票仅需确定2-4点关键因素，只有这样才能比对手挖得更深。

**SpaceX应用示例**:
| 关键因素 | 论证深度 | 数据支撑 |
|---------|---------|---------|
| 成本效益临界点突破 | 猎鹰9号复用成本模型 | 10次复用后成本下降70% |
| 产业链价值重估 | 卫星服务占52.91%份额 | GMI市场规模数据 |
| SpaceX IPO催化 | 估值锚确立 | 2026年Q2预期 |

**写作要求**: 报告首页用20%以上篇幅阐明这2-4点关键因素。

### 1.2 做出比一致预期更准确的预测

**方法论**: 
- 自建复杂模型（非简单引用一致预期）
- 每次模型调整做记录（you can't manage what you don't measure）
- 参考StarMine评估预测准确度

**SpaceX示例**:
> "星舰稳态发射1000次/年，单次边际运营成本~200万美元，毛利率~90%"
> ——这是原创预测，非市场一致预期

### 1.3 得出Superior目标价

**新手 vs 老手区别**:
- **新手**: 给乐观盈利预测×高市盈率
- **老手**: 研究为什么现在市场给这样的市盈率，过去股价波动原因是什么

**方法论**:
1. 了解股票的相对和绝对"峰""谷"
2. 研究过去三轮经济周期的平均市盈率
3. 参考可比公司，对不相关公司和行业做深入研究

### 1.4 评估一致预期，等待催化剂

**核心工作**: 持续了解市场一致预期，知道自己的杀手锏在哪里。

**催化剂三要素**:
1. Superior盈利预测
2. Superior估值方法
3. Superior投资者情绪预测

**SpaceX催化剂设计**:
> "SpaceX IPO将确立估值锚，类比特斯拉IPO对电动车板块的影响"

### 1.5 简单明了地推荐股票

**ENTER框架**（沟通价值五维度）:
| 维度 | 含义 | SpaceX应用 |
|------|------|-----------|
| **E**xpectational | 催化剂是什么，为什么会被市场认可 | SpaceX IPO确立估值锚 |
| **N**ovel | 市场不知道的是什么 | 卫星服务才是价值皇冠（52.91%） |
| **T**horough | 扎实的研究支持 | NASA报告+工程模型+历史数据 |
| **E**xaminable | 能指导决策的信息 | 具体标的筛选（SATS/ASTS等） |
| **R**evealing | 风险有哪些，把握有多大 | 四维风险矩阵+对冲表达 |

### 1.6 人际及组织关系

**时间分配**（"三个1/3"原则）:
- 1/3时间：调研上市公司、写报告、读资料
- 1/3时间：报告编辑、行文表述、逻辑推理
- 1/3时间：与基金经理等机构客户交流沟通

## 二、研报写作黄金标准（麦肯锡金字塔原理）

### 2.1 首页设计（20%以上时间）

**首页必须回答的问题**:
1. **行业/公司成功的逻辑是什么？**（核心投资逻辑）
2. **市场错在哪里？**（预期差）
3. **催化剂是什么？**（何时兑现）
4. **风险有哪些？**（把握有多大）

**SpaceX首页结构**:
```
标题：走向深空经济时代的"铁路"——商业航天产业深度研究

核心观点（区别于市场）:
1. 历史视角：商业航天=工业革命铁路体系（定量类比）
2. 成本效益：可回收火箭突破临界（LEO $50,000→$3,000/kg）
3. 价值重估：卫星服务才是皇冠（52.91% vs 发射2.46%）

投资建议：
优选"SpaceX合作+服务环节"标的：SATS、ASTS、TSAT、PL、LUNR

催化事件：SpaceX IPO（2026年Q2预期）

风险提示：技术/市场/政策/资本市场四维风险
```

### 2.2 正文结构（金字塔原理）

**原则**: 
- 开门见山表明结论
- 给出核心逻辑和原因
- 论证过程中观点不要交叉和重叠

**四幕剧结构**:
```
第一幕：认知颠覆（让读者意识到"之前的理解是错的"）
第二幕：逻辑验证（用数据证明拐点已至）
第三幕：价值发现（揭示预期差，筛选标的）
第四幕：行动指引（具体投资组合+风险对冲）
```

### 2.3 段落写作标准

**每段结构**: 
**论点→论据→论证→结论**

**SpaceX示例**:
> **论点**: 商业航天正处于成本效益临界点
> **论据**: LEO发射成本从航天飞机时代$50,000/kg降至商业火箭$3,000/kg
> **论证**: NASA《The Recent Large Reduction in Space Launch Cost》报告数据+猎鹰9号复用模型
> **结论**: 商业航天开始具备基础设施属性

## 三、行业分析框架（中信证券标准）

### 3.1 需求分析框架

**全景经济分析**:
- 找出影响行业销售的特定宏观经济变量
- 例：墨西哥水泥需求增长率是GNP增长率的1.7倍

**行业生命周期分析**:
| 阶段 | 特征 | 研究重点 |
|------|------|---------|
| 初创期 | 市场接受度怀疑，高风险 | 技术可行性、市场教育成本 |
| 成长期 | 产品被接受，销售加速 | 渗透率、新用户群体 |
| 成熟期 | 与经济趋势同步 | 市场份额、成本控制 |
| 衰退期 | 需求逐步减少 | 转型机会、剩余价值 |

**客户研究**:
- 将客户划分入不同子市场
- 研究影响每个子市场需求的因素
- 用不同划分标准进行交叉验证

### 3.2 供给分析框架

**供给能力计算**:
```
供给能力 = 本地产能 + 未来3-5年计划增加的产能 - 关闭产能
```

**产能利用率判断**:
- 供给<需求：价格上升，盈利能力增强
- 供给=需求：市场均衡，盈利按趋势线发展
- 供给>需求：价格走低，限制行业投资

### 3.3 获利能力分析

**影响定价的因素**:
1. **产品差别性**：差异化程度越高，定价权越强
2. **行业进入难易**：壁垒越高，价格波动越小
3. **行业集中度**：集中度越高，价格越稳定
4. **原材料成本**：成本转嫁能力决定盈利稳定性

### 3.4 竞争分析（波特五力模型）

```
        供应商议价能力
                ↑
潜在竞争者 → [行业竞争者] → 替代品威胁
                ↓
        客户议价能力
```

**四步分析**:
1. 细分行业，明确目标公司所在细分
2. 识别竞争对手
3. 评价竞争对手战略和影响
4. 概括竞争双方优劣势

## 四、SpaceX研报的独特方法论

### 4.1 历史类比法（区别于常规框架）

**核心创新**: 不直接分析行业，而是找到历史参照系。

**类比设计**:
| 维度 | 铁路时代 | 商业航天时代 | 数据支撑 |
|------|---------|-------------|---------|
| 技术突破 | 蒸汽机效率突破 | 可回收火箭技术 | 成本数量级下降数据 |
| 成本下降 | 0.48便士→<1美分/吨英里 | $50,000→$3,000/kg | LEO发射成本统计图 |
| 经济影响 | 跨区域分工 | 卫星互联网 | 星链用户增长数据 |
| 发展阶段 | 早期：低频次高成本 | 规模化前夕 | 发射频率趋势图 |

**论证技巧**: 反事实分析
> "若假设铁路未出现，美国农业土地价值将下降50%"
> → 证明铁路创造了需求，而非仅仅满足需求

### 4.2 "三阶导"企业分析模型

**创新点**: 用数学隐喻描述业务演进。

```
一阶导：基础业务（发射服务）
├─ 作用：建立运力优势
├─ 指标：发射频率、复用次数
└─ 价值：支撑作用

二阶导：核心业务（卫星服务）
├─ 作用：将运力转化为收入
├─ 指标：用户数、ARPU、覆盖率
└─ 价值：利润增长引擎

三阶导：远景业务（太空算力）
├─ 作用：重塑估值逻辑
├─ 指标：1TW算力部署、GPU数量
└─ 价值：未来定位
```

### 4.3 价值重估话语体系

**市场共识 vs 作者观点**:
- **市场**: 发射业务是核心（关注度最高）
- **作者**: 发射仅占2.46%，卫星服务才是皇冠（52.91%）
- **预期差**: 市场过度关注火箭，低估服务环节

**论证工具**:
- 市场份额饼图（直观展示占比差异）
- 价值链分析表（各环节价值分布）
- 投资建议映射（服务环节标的筛选）

### 4.4 风险对冲表达

**高阶技巧**: 风险不是罗列，而是与投资建议一一对应。

```
投资建议: 优选"SpaceX合作+服务环节"标的
对应风险:
├─ SpaceX上市进展不及预期（催化风险）
├─ 卫星发射进程不及预期（业务风险）
├─ 商业航天竞争格局加剧（竞争风险）
└─ 测算模型与假设偏差（方法论风险）
```

**效果**: 展示"我们已经考虑了风险"，增强可信度。

## 七、常见误区与避免方法

### 误区1：堆砌数据，没有观点
**错误**: 罗列大量数据，读者不知道核心观点是什么。
**正确**: 每个数据都服务于论证，首页明确2-4点关键因素。

### 误区2：技术崇拜，忽视商业价值
**错误**: 详细描述技术原理，没有回答"能不能挣钱"。
**正确**: 技术细节只说明"成本效益"和"商业化可行性"。

### 误区3：没有立场，回避判断
**错误**: "行业有好有坏，投资者自行判断"。
**正确**: 明确看多/看空，给出具体标的和催化剂。

### 误区4：忽视叙事，流水账罗列
**错误**: 按照时间顺序或业务板块罗列信息。
**正确**: 构建"认知颠覆→验证→发现→行动"叙事弧线。

### 误区5：风险堆砌，没有重点
**错误**: 罗列10+条风险，让读者感到无所适从。
**正确**: 风险与投资建议一一对应，形成对冲表达。

### 误区6：过度使用"辩证法"
**错误**: "一方面...另一方面..."，观点模糊。
**正确**: 明确立场，用数据和逻辑支撑判断。

## 八、应用到国产CPU研报的升级点

### 8.1 立场升级

**原表述**: "国产CPU行业分析"
**升级表述**: "国产CPU：半导体领域的'两弹一星'工程——从政策驱动到技术突破的历史性跨越"

### 8.2 核心类比升级

**历史坐标**: 
- 国产CPU当前阶段 = 1950s-60s日本汽车产业
- 政策保护 → 技术模仿 → 自主创新 → 全球竞争
- 石油危机催化 = AI算力需求催化

**反事实论证**:
> "若美国持续扩大芯片封锁，中国数字经济年损失将达X亿元，国产CPU自主可控是必由之路"

### 8.3 "三阶导"模型升级

```
一阶导：基础产品（海光/兆芯/龙芯/飞腾/鲲鹏/申威）
├─ 作用：建立自主可控能力
└─ 指标：性能参数、市场份额

二阶导：生态建设（统信UOS/麒麟OS/软件适配）
├─ 作用：从"能用"到"好用"
└─ 指标：适配软件数量、用户体验

三阶导：AI算力融合（CPU+GPU+NPU异构计算）
├─ 作用：重塑估值逻辑
└─ 指标：AI推理性能、算力集群规模
```

### 8.4 预期差升级

**市场共识**: "国产CPU性能差、生态弱、无法商用"
**作者观点**: "国产CPU性价比已达进口95%，灵生超算验证技术突破，信创市场强制替代"
**预期差**: 市场仍停留在"国产替代政策博弈"阶段，忽视"技术成长"逻辑

### 8.5 催化剂升级

**新增催化**:
1. **灵生超算发布**（2025年底）：国产CPU算力世界领先，技术突破验证
2. **信创三期政策**（2026Q2预期）：订单放量
3. **海光五号发布**（2026H2）：性能再突破
4. **央企100%信创考核**（2027）：强制替代
