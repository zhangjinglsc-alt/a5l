### 适用场景

| 场景 | 使用建议 |
|------|----------|
| **新兴技术行业** | 重点使用历史类比法 + 成本效益临界分析 |
| **产业链复杂行业** | 重点使用分层拆解 + 价值链分析 |
| **快速成长行业** | 重点使用"多阶导"业务分析 + 催化识别 |
| **资本密集行业** | 重点使用工程模型验证 + 财务健康度筛选 |

### 前置条件

- [ ] 目标行业已有初步商业化迹象
- [ ] 可获取行业核心数据（市场规模、成本结构等）
- [ ] 行业内有代表性龙头企业可供分析
- [ ] 明确的投资主题或催化事件

## 研究流程详解

### 第一步：锚定产业本质（历史类比法）

#### 核心任务

将**{目标行业}**与历史上类似产业进行类比，回答："这个行业为什么重要？"

#### 操作步骤

**Step 1.1 选择历史类比对象**

| 目标行业特征 | 建议类比对象 | 核心可比性 |
|--------------|--------------|------------|
| 基础设施型技术 | 工业革命铁路 | 成本下降→网络效应→需求创造 |
| 平台型技术 | 互联网/移动互联网 | 网络效应→生态扩张 |
| 能源型技术 | 石油/电力革命 | 成本优势→替代加速 |
| 计算型技术 | 计算机/云计算 | 性能提升→应用爆发 |
| 生物型技术 | 绿色革命/疫苗 | 技术突破→大规模应用 |

**Step 1.2 构建类比维度**

```
┌─────────────────────────────────────────────────────────────┐
│                    历史类比分析模板                          │
└─────────────────────────────────────────────────────────────┘

【类比对象】: {历史产业名称} vs {目标行业名称}

┌─────────────────┬─────────────────────────┬─────────────────────────┐
│    类比维度      │      历史产业            │       目标行业          │
├─────────────────┼─────────────────────────┼─────────────────────────┤
│   技术本质       │ {历史产业的技术特征}      │ {目标行业的技术特征}     │
├─────────────────┼─────────────────────────┼─────────────────────────┤
│   成本逻辑       │ {成本下降路径}            │ {成本下降路径}           │
├─────────────────┼─────────────────────────┼─────────────────────────┤
│   经济影响       │ {对经济的改变}            │ {预期经济影响}           │
├─────────────────┼─────────────────────────┼─────────────────────────┤
│   发展阶段       │ {所处历史阶段}            │ {当前发展阶段}           │
├─────────────────┼─────────────────────────┼─────────────────────────┤
│   需求创造       │ {需求如何被创造}          │ {需求创造机制}           │
└─────────────────┴─────────────────────────┴─────────────────────────┘

【核心洞见】:
{类比得出的关键结论，用于打破对该行业的传统认知}
```

**Step 1.3 提取历史规律**

引用学术研究或权威报告，提取历史产业的核心发展规律：
- 技术成熟度曲线特征
- 成本下降的S曲线规律
- 需求释放的滞后性特征
- 竞争格局演变路径

**Step 2.2 构建定量测算模型**

根据行业特性选择合适的测算模型：

| 行业类型 | 推荐测算模型 | 关键参数 |
|----------|--------------|----------|
| 制造业 | 成本分解模型 | 原材料、人工、制造费用占比 |
| 服务业 | 单位经济模型 | CAC、LTV、毛利率 |
| 平台型 | 网络效应模型 | 节点数、连接数、边际成本 |
| 技术型 | 学习曲线模型 | 累计产量、成本下降率 |
| 基础设施 | 规模经济模型 | 产能利用率、固定成本摊薄 |
