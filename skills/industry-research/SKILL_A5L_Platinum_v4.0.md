## 第二部分：句式模板库（80+句式，即拿即用）

### 2.1 观点陈述句式（核心必用）

#### 【历史类比型】
```
句式：我们认为当下的[新兴产业]与[历史事件]的[发展过程]具有可比性。
变量：[新兴产业]=研究对象 [历史事件]=参照系 [发展过程]=关键阶段
示例：我们认为当下的商业航天产业与工业革命时代的"蒸汽机—铁路"发展历程具有可比性。
应用：TMT(AI/半导体)/新能源/新消费均可使用
```

#### 【本质定位型】
```
句式：我们认为[研究对象]真正的重要性，正体现在其为[领域]提供了新的"[核心能力]"。
变量：[研究对象]=公司/技术/产品 [领域]=应用场景 [核心能力]=价值主张
示例：我们认为商业航天真正的重要性，正体现在其为经济活动的空间扩展提供了新的"交通边界"。
应用：强调差异化价值，建立独特定位
```

#### 【阶段判断型】
```
句式：我们认为[研究对象]当前所扮演的角色，并非对[既有需求]的简单满足，而更可能处于"[A条件]先行、[B需求]随后释放"的早期阶段。
变量：[A条件]=技术/政策/基础设施 [B需求]=市场需求/用户习惯
示例：我们认为商业航天当前所扮演的角色，并非对既有太空经济需求的简单满足，而更可能处于"交通条件先行、需求随后释放"的早期阶段。
应用：判断行业所处阶段，指导投资时点
```

#### 【临界点突破型】
```
句式：[技术/产品]正是推动[领域]突破[临界点]的"临门一脚"。
变量：[临界点]=成本/性能/规模阈值
示例：可回收火箭技术正是推动商业航天突破成本效益临界点的"临门一脚"。
应用：强调关键突破，建立投资 urgency
```

#### 【皇冠定位型】
```
句式：[细分领域]是[大产业]产业链的价值"皇冠"。[时间]其市场规模为[数值]（占[比例]），而[对比项]仅有[数值]（占[比例]）。
示例：卫星服务是商业航天产业链的价值"皇冠"。2024年其市场规模为1767亿美元（占52.91%），而发射业务仅有82亿美元（占2.46%）。
应用：揭示预期差，引导投资方向
```

### 2.2 前瞻判断句式（趋势预测）

#### 【趋势预测型】
```
句式：往后看，[A驱动因素]、[B驱动因素]和[C驱动因素]或成为[对象]的增长支撑。
变量：ABC=技术/政策/市场/成本等驱动因素
示例：往后看，技术迭代、成本下降和政策支持或成为人形机器人产业的增长支撑。
应用：建立多维度增长逻辑
```

#### 【数据引用型】
```
句式：往后看，据[机构]预计，[时间范围][指标]为[数值]，[增长描述]。
变量：[机构]=GMI/IDC/高盛/麦肯锡等
示例：往后看，据GMI预计，2025年至2034年复合年增长率为6.7%。
应用：用权威数据支撑预测
```

#### 【情景展望型】
```
句式：往后看，随着[关键变量]的[变化趋势]，[研究对象]或[预期结果]。
示例：往后看，随着产能爬坡完成和成本曲线下移，固态电池的商业化落地或加速推进。
应用：建立因果链条，指导投资节奏
```

#### 【逻辑推导型】
```
句式：基于此，我们认为[核心观点]。
示例：基于此，我们认为储能系统的真正价值在于其为新能源消纳提供"时空平移"能力。
应用：承接前文论证，得出投资结论
```

### 2.3 论证展开句式（逻辑构建）

#### 【历史论证模块】
```
开篇引入：
句式：推动[领域]变化的，不仅是[表层因素]，还包括[深层因素]发生变化。
示例：推动经济结构变化的，不仅是单项技术突破，还包括交通体系的成本和组织方式发生变化。

历史回顾：
句式：回顾[历史时期]可以发现，[技术A]带来的影响包括[效果1]。而[技术B]体系成网之后，[效果2]进一步真正转化为[深层影响]。到[时间点]，[核心指标]已达到[数值]，形成[覆盖范围]。

历史数据论证：
句式：[时期]，[指标]从[数值A]降至[数值B]，[效果描述]。
示例：19世纪末，铁路货运的平均价格已降至不足1美分/吨英里。
```

#### 【反事实论证模块】
```
句式：基于反事实方法的测算显示，在[既定条件]下，若假设[条件未发生]，[对象]将出现[负面结果]。这一结果意味着，[技术/事件]并非[简单作用]，而是在相当程度上通过[机制]，主动创造[新结果]。

变量说明：
- [既定条件]=历史/经济/技术背景
- [条件未发生]=关键假设不成立
- [负面结果]=可量化的损失
- [机制]=作用路径
- [新结果]=新创造的价值

示例：
基于反事实方法的测算显示，在19世纪美国经济条件下，若假设铁路未出现，美国农业土地价值将下降50%。这一结果意味着，铁路的出现并非对既有运输需求的简单满足，而是在相当程度上通过创造新的跨区域贸易可能性，主动创造了新的经济价值。

应用：证明技术的"创造性破坏"价值，而非简单的需求满足
```

#### 【成本效益论证模块】
```
成本临界判断：
句式：一项革命性技术要实现大规模商业化，其根本前提在于跨越特定的[成本效益临界点]。

成本数量级下降：
句式：自[时期]以来，[指标]已经从[高数值]/[单位]以上，下降至[低数值]/[单位]量级。
示例：自航天飞机时代以来，LEO发射成本已经从5万美元/kg以上，下降至数千美元/kg量级。

成本下降影响：
句式：当[指标]出现[变化幅度]时，[对象]的[属性]随之发生变化——从[旧状态]转向[新状态]。
示例：当单位成本出现数量级下降时，航天活动的经济属性随之发生变化——从一次性、不可复制的工程任务，逐步转向可被反复调用的运输能力。
```

#### 【产业链论证模块】
```
产业链分层：
句式：整体看，[大产业]可分为[数量]大类型，分别是[A]、[B]、[C]、[D]和[E]。

价值占比分析：
句式：其中，[细分领域A]占[大产业]市场规模最大，[年份]其市场规模为[数值]（占[比例]%）。而[细分领域B]仅有[数值]（占[比例]%）。

价值链判断：
句式：结合我们在[章节]的梳理，[环节]上限或更高。
示例：结合我们在第三部分的梳理，服务端（尤其是卫星服务）上限或更高。
```

### 2.4 对比修辞句式（强调突出）

#### 【业务分层对比】
```
句式：相比于作为[定位A]的[业务A]和作为[定位B]的[业务B]，[业务C]代表了[公司]在[时代]的新定位：不单单要做[旧定位]，而是要成为[新定位]。

变量：
- [定位A]=基础/现金流/支撑
- [定位B]=核心/利润/增长
- [定位C]=远景/战略/重塑

示例：相比于作为基础业务的发射业务和作为核心业务的卫星服务，太空算力业务代表了SpaceX在人工智能时代的新定位：不单单要做太空运输商，而是要成为太空基础设施运营商。

应用：清晰展示公司业务演进路径，建立长期价值认知
```

#### 【优劣对比】
```
句式：[A]作为[属性]更为[优势]，而[B]作为[属性]更为[劣势]。
句式：相比于[旧事物]，[新事物]在[维度A]、[维度B]和[维度C]更具优势。
```

#### 【强调修辞】
```
数据冲击：
句式：[指标]：[从数值A]→[到数值B]，[效果]。
示例：LEO发射成本：从$50,000/kg→$3,000/kg，数量级下降。

极端表述：
句式：[对象]开创了[领域]的先河。
示例：SpaceX开创了高频发射节奏和可重复使用火箭的先河。

强调唯一性：
句式：[公司/产品]是[领域]的[唯一/首个/最]...
示例：SpaceX是全球领先的发射服务提供商。
```

#### 【"三阶导"分析框架】
```

句式：[公司]"一阶导"：[业务A]作为[定位]。作为[角色]，[公司]开创了[特征]。从[维度1]看，[分析1]；从[维度2]看，[分析2]。

变量：
- [定位]=基础/支撑/现金流
- [角色]=市场地位
- [特征]=核心能力
- [维度]=技术/成本/规模

二阶导（核心业务）：
句式：[公司]"二阶导"：[业务B]带来[效果]。公司通过[机制]，通过[手段]，提供[价值]。往后看，[驱动因素]或成为增长支撑。

变量：
- [效果]=收入/利润/市场份额
- [机制]=商业模式
- [手段]=技术/产品/服务
- [价值]=客户价值主张

三阶导（远景业务）：

### 2.6 投资建议句式（决策指导）

#### 【双维度分类】
```
句式：从[维度1]来看，[行业]或可分为[A方向]和[B方向]两个方向；从[维度2]来看，亦可以将[行业]分为[C类别]及[D类别]。往后看，建议优先关注"[优选组合]"组合的相关标的。

- [维度1]=技术路线/商业模式/应用场景

示例：从技术路线来看，国产CPU或可分为x86兼容和自主架构两个方向；从竞争格局来看，亦可以分为信创核心和差异化优势。往后看，建议优先关注"信创核心+自主可控"组合的相关标的。

2）[类别2]【标的4】【标的5】；

## 第三部分：细粒度写作标准（微观层面）

### 3.1 段落结构标准

**标准结构：观点→数据→论证→结论**

```
【观点】商业航天正处于成本效益临界点
    ↓
【数据】LEO发射成本从航天飞机时代$50,000/kg降至商业火箭$3,000/kg
    ↓
【论证】NASA《The Recent Large Reduction in Space Launch Cost》报告数据+猎鹰9号复用模型
    ↓
【结论】商业航天开始具备基础设施属性
```

**每段检查清单**:
- [ ] 开头有明确观点句
- [ ] 中间有数据/证据支撑
- [ ] 论证逻辑清晰无跳跃
- [ ] 结尾有结论或过渡
- [ ] 段落长度控制在150-300字

### 3.2 章节过渡技巧

**钩子设计（章节结尾）**:
```
功能：引导读者继续阅读下一章
句式："[关键问题]的答案，将在下一章揭晓..."
示例："卫星服务为何是产业链的价值皇冠？我们将在第二部分详细拆解..."
```

**承接逻辑（章节开头）**:
```
功能：建立与上章的连贯性
句式："承接[上章结论]，我们从[新视角]重新审视..."
示例："承接第一部分的成本效益分析，我们从产业链视角审视价值分配..."
```

**跨章节线索呼应**:
```
功能：强化叙事主线
技巧：关键概念（如"三阶导"）在多个章节重复出现，形成贯穿全文的线索
```

### 3.3 图表叙事功能

**图表标准结构**:
```
【数据呈现】→【趋势判断】→【意义提炼】
```

**图表标题句式**:
```
趋势图："[指标]变化趋势（[时间范围]）"
对比图："[对象A]与[对象B]的[维度]对比"
结构图："[产业]价值链分布（[年份]）"
```

**图表后解读逻辑**:
```
句式：从[图表]可以看出，[趋势描述]。这意味着[意义提炼]，进一步验证了[核心观点]。
示例：从LEO发射成本变化图可以看出，2010-2024年间成本下降超过90%。这意味着商业航天已突破成本效益临界点，进一步验证了我们关于行业基础设施化的判断。
```

## 第四部分：行业差异化研究框架

### 4.1 TMT行业研究框架

**核心维度**:
| 维度 | 研究重点 | 关键指标 |
|------|---------|---------|
| 技术迭代 | 代际更替节奏、技术成熟度 | 专利数量、研发支出占比 |
| 生态构建 | 开发者数量、应用丰富度 | APP数量、开发者收入 |
| 商业模式 | 变现路径、客户付费意愿 | ARPU、LTV/CAC |
| 网络效应 | 用户规模、双边市场效应 | 用户增长、平台GMV |

**特殊句式**:
```
技术迭代："[技术]正从[阶段A]向[阶段B]演进，[关键指标]提升[X]倍"
生态构建："[平台]已构建[数量]开发者、[数量]应用的生态系统"
商业模式："[公司]通过[模式]实现[指标]的变现效率"
```

### 4.2 医药行业研究框架

**核心维度**:
| 维度 | 研究重点 | 关键指标 |
|------|---------|---------|
| 临床管线 | 在研产品、临床进度 | 管线数量、临床阶段 |
| 审批路径 | 监管政策、获批概率 | FDA/NMPA审批时间 |
| 市场空间 | 适应症人群、渗透率 | TAM/SAM/SOM |
| 竞争格局 | 同类竞品、差异化优势 | 竞品进度、市场份额 |

**特殊句式**:
```
临床管线："[公司][产品]处于[阶段]，预计[时间]完成[里程碑]"
审批路径："基于[历史数据]，[产品]获批概率为[X]%，预计[时间]获批"
市场空间："[适应症]全球患者[数量]人，峰值销售有望达到[金额]"
```

### 4.3 消费行业研究框架

**核心维度**:
| 维度 | 研究重点 | 关键指标 |
|------|---------|---------|
| 品牌力 | 品牌认知、溢价能力 | 品牌搜索指数、毛利率 |
| 渠道力 | 渠道覆盖、终端动销 | 门店数量、坪效 |
| 产品力 | 产品矩阵、创新能力 | SKU数量、新品贡献率 |
| 用户运营 | 用户粘性、复购率 | MAU、复购率 |

**特殊句式**:
```
品牌力："[品牌]在[品类]心智份额[X]%，溢价能力较行业平均高[X]%"
渠道力："[公司]已覆盖[数量]终端，[渠道类型]占比[X]%"
产品力："[产品]上市[时间]内销售额[金额]，占公司收入[X]%"
```

### 4.4 制造行业研究框架

**核心维度**:
| 维度 | 研究重点 | 关键指标 |
|------|---------|---------|
| 成本优势 | 规模效应、工艺优化 | 单位成本、产能利用率 |
| 技术壁垒 | 专利保护、Know-how | 专利数量、良率 |
| 产能扩张 | 资本开支、产能爬坡 | CapEx、产能利用率 |
| 客户结构 | 大客户依赖、订单可见性 | 前五大客户占比 |

**特殊句式**:
```

技术壁垒："[技术]专利[数量]项，良率[X]%领先行业"
产能扩张："[项目]预计[时间]投产，新增产能[X]万吨"

### 4.5 周期行业研究框架

| 维度 | 研究重点 | 关键指标 |

库存周期："行业处于[补库/去库]阶段，预计持续[时间]"
```

## 第五部分：研报结构标准模板

### 5.1 标准结构（70+页）

```
封面页（1页）
├─ 标题（立场明确）
├─ 核心观点（3-4点）
├─ 投资建议摘要
└─ 免责声明+资质

投资聚焦（2-3页）← 首页20%篇幅
├─ 区别于市场的观点（3点）
├─ 2-4点关键因素阐述
├─ 近期催化
└─ 结论与建议

第一章：锚定产业本质（10-15页）
├─ 历史类比或核心框架
├─ 为什么现在重要？
├─ 地缘政治/宏观背景
└─ 本章小结+下章预告

第二章：验证商业化可行性（30-40页）
├─ 技术路线/商业模式对比
├─ 成本效益分析
├─ 产业链价值分布
├─ 核心企业深度（三阶导分析）
└─ 本章小结+下章预告

第三章：投资建议（15-20页）
├─ 催化事件识别
├─ 投资维度确定
├─ 重点标的筛选（8-10家）
├─ 投资组合建议
└─ 本章小结+下章预告

第四章：风险提示（2-3页）
└─ 与投资建议对应的风险

附录（5-10页）
├─ 图表目录
├─ 财务数据表
├─ 模型假设与参数
└─ 分析师承诺
```

| 附录 | 5-10页 | 图表、数据、模型 | 2% |

## 第六部分：质量控制与自检

### 6.1 分析师自检三问（最终检验）

**写完后必须回答**:

1. **基金经理能在3分钟内get到投资逻辑吗？**
   - 检验：请同事阅读首页，3分钟后复述核心观点
   - 标准：能准确说出2-4点关键因素+催化剂+预期差

2. **如果我是买方，会基于这份报告下注吗？**
   - 检验：假设自己是基金经理，是否有足够信心建仓
   - 标准：愿意用真金白银验证报告逻辑

3. **市场错在哪里？我的杀手锏是什么？**
   - 检验：明确区分市场共识与作者观点
   - 标准：能找到至少1个显著的EXPECTATIONAL GAP

**任一答案否定的，重写。**

### 6.2 常见误区检查清单

| 误区 | 错误表现 | 正确做法 |
|------|---------|---------|
| 堆砌数据 | 罗列大量数据，无明确观点 | 每个数据服务于论证 |
| 技术崇拜 | 详述技术原理，不答"能挣钱吗" | 技术只说明成本效益 |
| 没有立场 | "有好有坏，自行判断" | 明确看多/看空 |
| 流水账 | 按时间/板块罗列 | 构建叙事弧线 |
| 风险堆砌 | 罗列10+条风险 | 风险与建议对应 |
| 辩证法 | "一方面...另一方面..." | 明确立场，数据支撑 |

### 6.3 最终交付标准

| 维度 | 标准 | 检查方式 |
|------|------|---------|
| 立场鲜明 | 首页明确看多/看空 | 3秒判断 |
| 预期差清晰 | ENTER框架五维度 | 清单检查 |
| 关键因素 | 2-4点，数据支撑 | 首页20%篇幅 |
| 催化剂明确 | 时间/事件/影响清晰 | 可执行布局 |
| 风险对冲 | 与建议一一对应 | 逻辑闭环 |
| 数据验证 | 三源/二源一致 | 交叉验证 |
| 叙事完整 | 四幕剧结构 | 钩子设计检查 |
| 图表专业 | 85+张，叙事功能 | 图表自检清单 |

### 7.1 快速套用三步法
