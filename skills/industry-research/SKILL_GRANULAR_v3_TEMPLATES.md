# 研报写作句式模板库（SpaceX研报提炼版）

> 本模板库基于《全球商业航天深度研究系列（一）：SpaceX》逐段解析提炼
> 用途：快速构建研报框架、填充段落内容、保持一致文风

---

## 第一章：观点陈述句式

### 1.1 核心观点句（必用）

```
【历史类比型】
我们认为当下的[新兴产业]与[历史事件]的[发展过程]具有可比性。
→ 例：我们认为当下的人工智能产业与工业革命时代的"电力普及"具有可比性。

【本质定位型】
我们认为[研究对象]真正的重要性，正体现在其为[领域]提供了新的"[核心能力]"。
→ 例：我们认为低空经济真正的重要性，正体现在其为城市交通提供了新的"空间维度"。

【阶段判断型】
我们认为[研究对象]当前所扮演的角色，并非对[既有需求]的简单满足，而更可能
处于"[A条件]先行、[B需求]随后释放"的早期阶段。
→ 例：我们认为固态电池当前所扮演的角色，并非对现有电动车的简单满足，而更可能
处于"技术突破先行、规模应用随后释放"的早期阶段。

【临界点突破型】
[技术/产品]正是推动[领域]突破[临界点]的"临门一脚"。
→ 例：固态电解质技术正是推动锂电池突破能量密度临界点的"临门一脚"。
```

### 1.2 价值定位句

```
【皇冠定位】
[细分领域]：[大产业]产业链的价值"皇冠"。
→ 例：操作系统：智能汽车产业链的价值"皇冠"。
→ 例：储能系统：新能源产业链的价值"皇冠"。

【皇冠定位（含数据）】
[细分领域]是[大产业]产业链的价值"皇冠"。[时间]其市场规模为[数值]（占[比例]），
而[对比项]仅有[数值]（占[比例]）。
→ 例：卫星服务是商业航天产业链的价值"皇冠"。2024年其市场规模为1767亿美元
（占52.91%），而发射业务仅有82亿美元（占2.46%）。
```

### 1.3 前瞻判断句

```
【趋势预测型】
往后看，[A驱动因素]、[B驱动因素]和[C驱动因素]或成为[对象]的增长支撑。
→ 例：往后看，技术迭代、成本下降和政策支持或成为人形机器人产业的增长支撑。

【数据引用型】
往后看，据[机构]预计，[时间范围][指标]为[数值]，[增长描述]。
→ 例：往后看，据IDC预计，2025-2030年全球AI服务器出货量CAGR将达25%。

【情景展望型】
往后看，随着[关键变量]的[变化趋势]，[研究对象]或[预期结果]。
→ 例：往后看，随着产能爬坡和成本曲线下移，固态电池的商业化落地或加速推进。

【逻辑推导型】
基于此，我们认为[核心观点]。
→ 例：基于此，我们认为储能系统的真正价值在于其为新能源消纳提供"时空平移"能力。
```

## 第二章：论证展开句式

### 2.1 历史论证模块

```
【开篇引入】
推动[领域]变化的，不仅是[表层因素]，还包括[深层因素]发生变化。
→ 例：推动经济结构变化的，不仅是单项技术突破，还包括交通体系的成本和组织方式发生变化。

【历史回顾】
回顾[历史时期]可以发现，[技术A]带来的影响包括[效果1]。而[技术B]体系成网之后，
[效果2]进一步真正转化为[深层影响]。到[时间点]，[核心指标]已达到[数值]，
[具体描述]。
→ 例：回顾第一次工业革命可以发现，蒸汽机带来的影响包括提高生产效率。而铁路体系
成网之后，蒸汽动力进一步真正转化为推动经济结构变化的基础设施。到19世纪末，
美国铁路运营里程已达到16万英里，形成覆盖全国的运输网络。

【类比过渡】
这一历史逻辑，可以直接类比到今天的[领域]。当前[领域]的变化，同样并非源于
[表层原因]，而是[深层原因]。
→ 例：这一历史逻辑，可以直接类比到今天的商业航天。当前航天领域的变化，同样
并非源于火箭原理的根本突破，而是运输体系的组织方式发生改变。

【权威佐证】
[权威机构]在其[报告名称]中指出，[核心数据对比]。
→ 例：NASA在其官方技术报告《The Recent Large Reduction in Space Launch Cost》中指出，
单位发射成本已经从5万美元/kg以上，下降至数千美元/kg量级。

【阶段标志】
当[关键指标]出现[变化]时，[领域]的[属性]随之发生变化——从[A状态]，
逐步转向[B状态]。我们认为这正是[领域]开始具备[核心属性]的关键标志。
→ 例：当单位成本出现数量级下降时，航天活动的经济属性随之发生变化——从一次性、
不可复制的工程任务，逐步转向可被反复调用的运输能力。我们认为这正是商业航天
开始具备基础设施意义的关键标志。
```

### 2.2 质疑回应模块

```
【质疑呈现】
当前围绕[领域]的核心质疑之一，或在于"[质疑点1]"和"[质疑点2]"。
→ 例：当前围绕商业航天的核心质疑之一，或在于"现实需求规模有限"和"应用场景尚不成熟"。

【历史反驳】
但[历史参照物]的历史表明，[核心规律]。
→ 例：但铁路的历史表明，交通基础设施在完成商业化并实现成本系统性下降之前，
其潜在需求往往并不可见。

【逻辑推导】
正是在[条件A]和[条件B]之后，[结果1]、[结果2]才逐步出现。
→ 例：正是在铁路网络成型、运输成本显著下降之后，跨区域分工、规模化生产以及
全国性市场才逐步出现。

【结论重构】
因此，[研究对象]在当前阶段所扮演的角色，并非对[既有X]的简单满足，而更可能
处于"[A先行]、[B随后释放]"的早期阶段。我们认为其经济影响不应仅以[表面指标]
来衡量，而应结合[深层逻辑]来理解。
→ 例：因此，商业航天在当前阶段所扮演的角色，并非对既有太空经济需求的简单满足，
而更可能处于"交通条件先行、需求随后释放"的早期阶段。
```

### 2.3 数据引入模块

```
【机构引用】
据[权威机构]的[测算/统计/预计]，[时间范围][核心指标]为[具体数值]。
→ 例：据GMI的最新测算，全球太空经济市场在2024年的市场规模为4180亿美元。

【变化趋势】
自[时间节点]以来，[核心指标]已经从[A数值]，[变化方向]至[B数值]。
→ 例：自航天飞机时代以来，单位发射成本已经从5万美元/kg以上，下降至数千美元/kg量级。

【倍数关系】
[时间段]，[指标A]增长约[X]倍，[指标B]相当于[参照系]的约[Y]%。
→ 例：1850至1912年间，英国铁路年客运里程增长约16倍，铁路客运所产生的社会储蓄
相当于同期英国累计全要素生产率增长的约15%。

【对比呈现】
[指标]为[当前数值]，较[参照物]的[参照数值][变化描述]。
→ 例：燃煤成本为每吨英尺0.012便士，较同期马拉系统的0.48便士显著降低。

【极端数据】
[主体]已创下最高[指标]达[数值]的历史纪录。
→ 例：猎鹰9号一级助推器已创下最高复用32次的历史纪录。
```

### 2.4 结论推导模块

```
【意义解读型】
这一结果/数据/现象意味着，[研究对象]并非只是[表象功能]，而是在相当程度上
通过[核心机制]，[深层影响]。
→ 例：这一结果意味着，铁路并非只是服务既有经济需求，而是在相当程度上通过降低
运输与交易成本，主动创造了新的经济活动可能性。

【逻辑推断型】
由此推断/可见/可推，[研究对象]的[核心逻辑/规律]在于"[简洁概括]"。
→ 例：由此可推，可回收火箭降低成本的核心逻辑在于"以次数换成本"。

【综合结论型】
综上所述，[研究对象]所展现的[特征]，不仅在于[A层面]，更核心的价值在于其
依托[核心机制]所实现的[本质属性]。
→ 例：综上所述，可回收火箭所展现的经济效益，不仅在于对单次发射成本的重构，
更核心的价值在于其依托高复用率所实现的"规模经济"。

【意义升华型】
基于以上分析，我们认为[研究对象]真正的重要性，正体现在其为[活动]的[维度]扩展
提供了新的"[边界]"。
→ 例：基于以上分析，我们认为商业航天真正的重要性，正体现在其为经济活动的空间扩展
提供了新的"交通边界"。
```

## 第三章：对比修辞句式

### 3.1 成本对比

```
【降幅对比】
[指标]已经从[A数值]，下降至[B数值]，[降幅描述]。
→ 例：单位发射成本已经从5万美元/kg以上，下降至数千美元/kg量级。

【优劣对比】
[指标]为[当前数值]，较[参照物]的[参照数值][变化描述]。
→ 例：不锈钢成本约5美元/公斤，比碳纤维复合材料低约95%。

【临界点对比】
当[条件]时，[指标]可[变化A]。往后看，随着[驱动因素]，[变化B]。
→ 例：当复用次数超过10次时，单次发射平均成本可稳定在约1700万美元。往后看，
随着发射次数继续增加，单次成本降幅趋近70%。
```

### 3.2 份额对比

```
【主次对比】
[细分A]占[大市场]规模最大，[时间]其市场规模为[数值]（占[比例]）。
而[细分B]仅有[数值]（占[比例]）。
→ 例：卫星服务占商业航天市场规模最大，2024年其市场规模为1767亿美元（占52.91%）。
而发射业务仅有82亿美元（占2.46%）。

【关注度对比】
[高关注领域]受到市场较多关注，但[实际价值领域]或具备更高上限。
→ 例：发射业务受到市场较多关注，但卫星服务端（尤其是卫星服务）上限或更高。
```

### 3.3 发展阶段对比

```
【由A到B】
[领域]正通过[核心手段]实现从"[状态A]"向"[状态B]"的根本转变。
→ 例：商业航天正通过可回收火箭技术实现从"高成本、低频次"向"低成本、规模化"的根本转变。

【代际跨越】
[产品/技术]标志着[领域]性能的代际跨越。
→ 例：JUPITER 3卫星代表了当前GEO通信技术的巅峰，拥有高达500Gbps的通信容量，
标志着卫星宽带性能的代际跨越。
```

## 第四章：公司分析句式

### 4.1 公司简介

```
【公司定位】
[公司]由[创始人]于[时间]正式创立，公司以[使命]为核心使命。
→ 例：SpaceX由埃隆·马斯克于2002年正式创立，公司以降低太空探索成本、构建支持人类
在火星及其他行星上生活的必要基础设施为核心使命。

【业务定位】
[公司]是[领域]的[定位描述]，专注于[核心业务]。
→ 例：SpaceX是新时代的太空基础设施运营商，专注于运载火箭、载人航天器、卫星星座及
深空运输系统的研发、制造与发射服务。
```

### 4.2 发展历程

```
【阶段划分】
复盘[公司]的发展阶段，可分为[数量]个主要阶段：
- [时间段1]：[阶段特征]，完成了从[状态A]向[状态B]的跨越。
- [时间段2]：[阶段特征]，建立了以[核心能力]为基础的商业闭环。
- [时间段3]：[阶段特征]，实现了从[模式A]向[模式B]的升维。

→ 例：
复盘SpaceX的发展阶段，可分为三个主要阶段：
- 2002-2009：初创期，完成了从边缘探索者向国家航天梯队的跨越。
- 2010-2018：技术突破期，建立了以可回收技术为基础的商业闭环。
- 2019至今：生态升维期，实现了从"太空搬运工"向"平台订阅商"的升维。

【关键节点】
[公司]的重要节点包括[数量]个，主要包括[时间1]完成"[标志性事件]"，到[时间2]通过
[技术]重塑[领域]，再到[时间3]由[产品]开启[新阶段]。
```

### 4.3 业务拆解（三阶导框架）

```
【框架引入】
因此，在后续的业务拆解中，我们将其核心业务分成"三阶导"：
- 一阶导（基础业务）：[业务A]作为[定位]
- 二阶导（核心业务）：[业务B]带来[效果]
- 三阶导（远景业务）：[业务C]代表[新定位]

→ 例：
- 一阶导（基础业务）：发射业务作为传统支撑
- 二阶导（核心业务）：卫星业务带来利润增长
- 三阶导（远景业务）：太空算力代表新定位

【产品分析】
[产品]是[公司]当前的[定位描述]，核心竞争力在于[核心优势]。
从技术参数看，[具体参数]。
从经营数据看，[营收/利润数据]。
往后看，[预期判断]。
```

### 4.4 竞争优势

```
【三大优势】
[公司]已经通过[战略]，系统性地构建了[优势A]、[优势B]以及[优势C]三大核心优势。
→ 例：SpaceX已经通过多基地战略协同高度垂直一体化的地面发射设施，系统性地构建了
地理布局弹性、技术赋能效率以及研发迭代敏捷三大核心优势。

【核心能力】
[公司]的核心增长逻辑是通过[能力]带动[结果]，这种[能力]正推动公司向[方向]迈进。
→ 例：SpaceX的核心增长逻辑是通过技术迭代带动工程放量，这种底层能力的积累正推动
公司向更深远的航天生态迈进。
```

## 第五章：投资建议句式

### 5.1 投资主线

```
【维度划分】
围绕当下的[因素A]和[因素B]，投资主线或可通过[数量]个维度进行衡量。
从[维度A]来看，[分类方式1]；从[维度B]来看，[分类方式2]。
→ 例：围绕当下的产业特征和后续潜在催化，投资主线或可通过两个维度进行衡量。
从产业特征来看，商业航天板块可分为硬件供应商和服务供应商两个方向；
从竞争格局看，亦可以分为SpaceX产业链及差异化优势公司。

【两维四象限】
结合这两点进行分类，我们认为在[象限描述]领域的公司，具备叠加"[组合特征]"
的双重利好，或在[催化剂]背景下，更值得关注。
→ 例：结合这两点进行分类，我们认为"SpaceX合作+服务环节高上限"领域的公司，
具备叠加"龙头合作+高价值量"的双重利好，或在SpaceX上市背景下，更值得关注。
```

### 5.2 具体建议

```
【分层建议】
建议关注：
1）优选"[特征1]"相关公司[代码列表]；
2）占据[特征2]优势生态位[代码列表]；
3）差异化[特征3][代码列表]。

→ 例：
建议关注：
1）优选"SpaceX合作+服务环节高上限"相关公司【SATS】【ASTS】【TSAT】；
2）占据卫星服务优势生态位【GSAT】【IRDM】；
3）差异化发射业务【RKLB】【FLY】。
```

### 5.3 风险提示

```
【风险列举】
1）[风险类型1]风险：[具体描述]。若[触发条件]，将直接[影响1]。这不仅会[影响2]，
还可能导致[影响3]。
→ 例：卫星发射进程不及预期风险：商业航天高度依赖高频次的发射支撑。若因技术瓶颈、
供应链中断或天气影响导致发射推迟，将直接拖累星座组网速度。这不仅会延缓商业化
服务落地，还可能导致企业错失轨道频率资源占用的先机。

2）[风险类型2]风险：[具体描述]。若[条件]，可能导致[结果]。
→ 例：技术发展不及预期风险：由于相关领域尚处于高投入、高风险的探索阶段，若技术突破
速度无法匹配资本市场的商业化预期，可能导致行业估值承压。
```

## 第六章：章节过渡句式

### 6.1 章首承接

```
【标准承接】
[时间]以来，以[标志性事件]为标志，[行业]得到[市场反应]。我们认为[核心判断]。
在和[受众]的交流中，我们发现[市场分歧]。
本章节我们将从[视角]出发，以[参照对象]为[方法]，[本章目标]。
→ 例：2026年以来，以美国SpaceX上市为标志，全球商业航天产业链得到较大的市场关注。
我们认为年内商业航天的产业趋势明确且重要。在和投资者的交流中，我们发现市场对于
商业航天的价值定位仍有分歧。本章节我们将从历史视角出发，以工业革命时代的铁路为
比较对象，解读商业航天对于经济发展的重要意义。

【方法递进】
作为[对象属性]，在结合[上章方法]之后，仍需要[本章方法]。本章节，我们通过[具体方法]，
来[本章目标]。
→ 例：作为新兴产业，在结合历史视角定性看其战略定位之后，仍需要定量测算产业空间
和各环节的价值分布。本章节，我们通过梳理全球商业航天产业链的组成和各环节的价值量
分布，来衡量以SpaceX为代表的商业航天公司在未来的产业发展前景。
```

## 第七章：强调与修辞句式

### 7.1 排比强调

```
【三大X】
[公司/领域]已经构建了[A维度]、[B维度]以及[C维度]三大核心[属性]。
→ 例：SpaceX系统性地构建了地理布局弹性、技术赋能效率以及研发迭代敏捷三大核心优势。

【螺旋上升】
[主体]始终遵循"[阶段A]—[阶段B]—[阶段C]—[阶段D]"的螺旋上升路径。
→ 例：SpaceX始终遵循"单点技术突破—工程化降本—规模化变现—利润回补研发"的
螺旋上升路径。

【四大/三大优势】
相比于[参照物]，[研究对象]具有[数量]大核心优势：第一是[优势1]，第二是[优势2]，
第三是[优势3]...
→ 例：相比于其他航空航天材料，不锈钢具有四大核心优势：第一是经济性，第二是热管理，
第三是耐久性，第四是制造便捷性。
```

### 7.2 极端强调

```
【极端数据】
[主体]已创下最高[指标]达[数值]的历史纪录。
→ 例：猎鹰9号一级助推器已创下最高复用32次的历史纪录。

【极端降幅】
[指标]降幅[超过/趋近][X]%。
→ 例：单次成本降幅高达约68%。/ 单次成本降幅趋近70%。

【极端场景】
[公司]完成了"由死转生"，从[绝境]到[重生]的转变。
→ 例：2008年完成"由死转生"，从边缘探索者向国家航天梯队的转变。

【彻底重构】
这种转变使得[核心]从"[旧模式]"转向了"[新模式]"。
→ 例：这种转变使得航天发射的经济核心从"如何降低制造单价"转向了"如何提升资产复用率"。
```

### 7.3 隐喻表达

```
【铁路/基础设施隐喻】
[领域]：走向[新时代]的"铁路"——为[活动]的[维度]扩展提供了新的"[边界]"。
→ 例：商业航天：走向深空经济时代的"铁路"——为经济活动的空间扩展提供了新的"交通边界"。

【皇冠隐喻】
[细分领域]：[大产业]产业链的价值"皇冠"。
→ 例：卫星服务：商业航天产业链的价值"皇冠"。

【临门一脚隐喻】
[技术]正是推动[领域]突破[临界点]的"临门一脚"。
→ 例：可回收火箭正是推动太空运力突破成本效益临界点的"临门一脚"。

【现金牛隐喻】
[业务]：当前确定性强的"现金牛"。
→ 例：星链业务：当前确定性强的"现金牛"。

【代际跨越隐喻】
标志着[领域]的代际跨越。
→ 例：标志着卫星宽带性能的代际跨越。
```
