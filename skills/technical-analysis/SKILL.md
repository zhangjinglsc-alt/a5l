# 技术分析 SKILL

## 描述

技术分析工具，用于数据分析、技术指标计算、图表生成、交易信号识别，适用于股票技术分析、期货分析、加密货币分析。

## 使用方法

触发此 Skill 的指令：
- `技术分析` - 启动技术分析
- `指标计算` - 计算技术指标
- `图表生成` - 生成技术图表

## 技术指标

### 趋势指标
- **移动平均线 (MA)** - 简单/指数/加权移动平均
- **MACD** - 异同移动平均线
- **布林带 (Bollinger Bands)** - 价格波动区间
- **ADX** - 平均趋向指数

### 动量指标
- **RSI** - 相对强弱指标
- **KDJ** - 随机指标
- **Williams %R** - 威廉指标
- **CCI** - 商品通道指数

### 成交量指标
- **OBV** - 能量潮
- **成交量MA** - 成交量均线
- **MFI** - 资金流量指标

### 波动率指标
- **ATR** - 真实波动幅度均值
- **布林带宽度** - 波动率测量

## 图表类型

- K线图
- 趋势线图
- 成交量图
- 多指标组合图

## 信号识别

- 金叉/死叉
- 背离信号
- 突破信号
- 形态识别
