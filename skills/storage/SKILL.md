# 存储 SKILL

## 描述

存储产业分析工具，覆盖DRAM、NAND Flash、NOR Flash、HBM（高带宽内存）、存储模组、存储控制芯片等，适用于存储产业投资分析、周期判断、概念股筛选。

## 使用方法

触发此 Skill 的指令：
- `存储` - 存储产业分析
- `DRAM` - DRAM分析
- `NAND` - NAND Flash分析
- `HBM` - 高带宽内存分析
- `存储芯片` - 存储芯片分析
- `存储周期` - 存储周期判断

## 覆盖领域

### 存储芯片类型
- **DRAM** - 动态随机存取存储器
- **NAND Flash** - 闪存
- **NOR Flash** - 代码存储
- **HBM** - 高带宽内存（AI服务器用）
- **SRAM** - 静态随机存取存储器

### 产业链
- 存储晶圆制造
- 存储模组
- 控制芯片
- 封测服务

### 应用领域
- 服务器/数据中心
- 智能手机
- PC/笔记本
- 汽车电子
- AI训练/推理

## 周期分析

### 存储周期特征
- 3-4年一轮周期
- 供需关系驱动
- 价格波动剧烈
- 资本开支周期

### 周期判断指标
- 现货价格走势
- 原厂产能利用率
- 库存水平
- 下游需求
