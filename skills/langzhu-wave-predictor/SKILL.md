# 🌊 浪主波浪理论预测系统 SKILL

## 描述

**A5L自我进化的波浪理论预测系统**

通过一天两次的预测-验证循环，不断学习和优化浪主波浪理论分析方法：
- **上午9:25** - 早盘预测
- **中午11:35** - 验证早盘 + 下午预测
- **下午15:05** - 验证总结

系统自动记录每次预测，对比实际走势，积累数据后优化判断逻辑和参数。

## 使用方法

### 手动触发
```bash
# 早盘预测
python3 skills/langzhu-wave-predictor/scripts/run_morning.sh

# 午盘验证+下午预测
python3 skills/langzhu-wave-predictor/scripts/run_noon.sh

# 收盘验证
python3 skills/langzhu-wave-predictor/scripts/run_afternoon.sh
```

### 查看报告
```bash
# 查看今日预测
ls skills/langzhu-wave-predictor/logs/report_*.md

# 查看日报
ls skills/langzhu-wave-predictor/logs/daily_summary_*.md
```

### 触发词（计划）
- `/浪主预测` - 查看最新预测
- `/预测日报` - 查看今日总结
- `/验证预测` - 手动验证待验证预测

## 预测-验证循环

```

                │
                │
11:35 ────────┴──> 验证早盘 ────> 下午预测 ────> 记录到数据库
                                          │
                                          │
15:05 ────────────────────────────────────┴──> 验证下午 ────> 生成日报
```

## 验证标准

| 预测 | 验证标准 | 准确度计算 |
|:-----|:---------|:-----------|
| 看涨(up) | 实际上涨 > 0 | min(涨幅%/2, 1.0) |
| 看跌(down) | 实际下跌 < 0 | min(跌幅%/2, 1.0) |
| 震荡(consolidation) | 涨跌幅 < 0.5% | 正确=0.8, 错误=0.3 |

## 注意事项

1. **预测仅供参考**，不构成投资建议
2. **数据延迟** - 实时数据可能有分钟级延迟
3. **模型局限** - 波浪理论有主观性，需结合实际判断
4. **学习周期** - 需要积累足够验证数据后才能优化
