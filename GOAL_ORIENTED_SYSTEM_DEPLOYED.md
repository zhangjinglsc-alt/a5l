## 💡 使用示例

### 查看当前进度
```bash
python3 TOOLS/goal_manager.py
# 输出: 进度看板 + 每日简报
```

### 运行完整进化周期
```bash
python3 TOOLS/integrated_evolution_system.py
# 输出: 4阶段完整报告
```

### 手动更新任务进度
```python
from goal_manager import GoalManager
gm = GoalManager()
gm.update_task_progress("T20260501...", 75, "核心逻辑开发中")
```

### 标记任务阻塞
```python
gm.block_task("T20260501...", "需要确认修复策略")
# Agent会立即报告，等待人类决策
```

### 完成任务并自评
```python
gm.complete_task("T20260501...", 6, 95, "自修复功能已实现")
# Agent会请求验收
```
