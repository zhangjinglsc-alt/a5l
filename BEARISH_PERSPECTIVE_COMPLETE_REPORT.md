## 💡 核心能力

### 1. 风险自查清单 (6大维度24项检查)

| 维度 | 检查内容 | 示例问题 |
|------|----------|----------|
| **估值风险** | PE/PB/PEG/历史分位 | 当前PE是否高于行业平均50%以上? |
| **技术风险** | 超买/背离/支撑/趋势 | RSI是否超过70(超买)? |
| **基本面风险** | 业绩/负债/现金流/减持 | 净利润增长率是否连续两季下降? |
| **市场风险** | 大盘/板块/VIX/政策 | 大盘是否处于下跌趋势? |
| **流动性风险** | 成交额/换手率/解禁 | 日均成交额是否低于1亿? |
| **集中度风险** | 客户/供应商/产品依赖 | 前五大客户收入占比是否超过50%? |

### 2. 策略逻辑审查 (6维度反驳式验证)

| 维度 | 空方论点示例 | 严重程度 |
|------|-------------|----------|
| **逻辑** | 突破新高但未考虑是否为假突破 | 🟠 高 |
| **数据** | 业绩增长数据与估值提升不匹配 | 🔴 致命 |
| **市场** | 大盘处于调整期，个股难以独善其身 | 🟡 中 |
| **时机** | 即将面临业绩披露期，不确定性增加 | 🟠 高 |
| **仓位** | 计划仓位过重，单票风险集中度过高 | 🟡 中 |
| **退出** | 未设定明确的止损和止盈点 | 🟠 高 |

### 3. 风险评分系统 (0-100)

```
🔴 70-100分: 极高风险 - 强烈建议观望
🟠 50-69分:  高风险   - 谨慎参与
🟡 30-49分:  中等风险 - 控制仓位
🟢 10-29分:  低风险   - 可以参与
⚪ 0-9分:    安全     - 按计划执行
```

**评分维度**:
- 风险项概率 × 影响 = 单项风险评分
- 策略缺陷数量 × 严重程度 = 缺陷扣分
- 综合加权 = 总体风险评分

## 🎮 使用示例

### 示例1: 生成风险自查清单
```python
from skills.ARCHITECT-5L-SUPER.SKILL import Architect5LSuperSkill

skill = Architect5LSuperSkill()

# 生成Markdown格式的自查清单
checklist_path = skill.generate_risk_checklist("300308.SZ")

```

**输出**: `risk_checklist_300308.md`

内容包含:
- 6大风险维度
- 24项检查点
- 勾选框格式
- 风险等级对照表

### 示例2: 空方视角全面分析
```python
# 持仓信息
position_info = {
    'planned_position': 0.20,  # 计划20%仓位
    'entry_price': 145.0,
    'stop_loss': 130.0,
    'take_profit': 180.0
}

# 策略逻辑
strategy_logic = """
买入逻辑:
1. CPO赛道龙头，业绩高增长
2. 股价突破历史新高，趋势强劲
3. AI算力需求爆发，行业景气度高
计划: 买入20%仓位，止损-10%，止盈+25%
"""

# 执行空方分析
report = skill.bearish_analysis(
    symbol="300308.SZ",
    position_info=position_info,
    strategy_logic=strategy_logic
)

print(f"风险评分: {report['overall_risk_score']}/100")
print(f"风险等级: {report['risk_level']}")
print(f"建议: {report['recommendation']}")
```

**输出**:
```
风险评分: 100.0/100
风险等级: 极高风险
建议: 强烈建议观望，当前风险过高，不宜入场。如已持仓，考虑减仓避险。
```

### 示例3: 审查策略逻辑
```python
# 单独审查策略逻辑
flaws = skill.review_strategy_logic("300308.SZ", strategy_logic)

for flaw in flaws:
    severity = "🔴" if flaw['severity'] == 'critical' else "🟠"
    print(f"{severity} [{flaw['aspect']}] {flaw['bearish_argument']}")
    print(f"   建议: {flaw['suggestion']}")
```

**输出**:
```

   建议: 等待回踩确认或放量突破后再入场
```
